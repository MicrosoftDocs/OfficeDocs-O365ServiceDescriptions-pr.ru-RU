# <a name="visio-for-the-web"></a>Visio для Интернета

Visio для Интернета расширяет возможности Microsoft Visio в веб-браузере, где вы можете работать с диаграммами непосредственно на веб-сайте, где хранится схема. Пользователи Microsoft Office 365 с Office в Интернете могут просматривать схемы и делиться ими, а также добавлять комментарии в дороге.
  
## <a name="apply-rich-formatting-to-text-and-shapes"></a>Применение форматирования к тексту и фигурам
<a name="BM_1"> </a>

Недоступно в Visio для Интернета. Функции редактирования, такие как дополнительные цвета, градиенты, эффекты и стили, доступны в классическом приложении Visio. С помощью Visio для Интернета можно добавить текстовое поле или выбрать из галереи фигур и применить стили, которые определяют эффекты заливки, контура и тени.
  
## <a name="arrange-objects"></a>Arrange objects
<a name="BM_2"> </a>

Пока недоступно в Visio для Интернета. В классическом приложении Visio вы можете перемещать, масштабировать, поворачивать и упорядочивать фигуры и текстовые поля, а также отменять группировку фигур, чтобы работать с ними по отдельности. 
  
## <a name="build-mashup-solutions"></a>Создание гибридных решений
<a name="BM_3"> </a>

Приложение Visio для гибридного API веб-приложения JavaScript позволяет разработчикам получать доступ к веб-схеме Visio, ее страницам и ее фигурам, а также управлять ими. Дополнительные сведения можно найти [в статье Visio for Web: вопросы](https://support.office.com/article/visio-online-frequently-asked-questions-e6647040-2fca-42ec-9fa5-d16a4e39e0ee)и ответы.
  
## <a name="create-diagrams"></a>Создание схем
<a name="BM_4"> </a>

Вы не можете создавать схемы в Visio для Интернета; Однако вы можете создавать схемы в классическом приложении Visio, а затем отправлять их в SharePoint или OneDrive для бизнеса. Но вы можете создавать схемы в классическом приложении Visio, отправлять их в SharePoint или OneDrive для бизнеса, а затем делиться ими.
  
## <a name="cut-copy-and-paste"></a>Команды "Вырезать", "Копировать" и "Вставить"
<a name="BM_5"> </a>

Пока недоступно в Visio для Интернета. Вы можете перемещать и удалять фигуры и текст в схеме с помощью команд "Вырезать", "Копировать" и "Вставить" классического приложения Visio.
  
## <a name="edit-diagrams"></a>Редактирование схем
<a name="BM_6"> </a>

Вы не можете вносить изменения в схему в Visio для Интернета. Чтобы внести изменения в схему, используйте классическое приложение Visio.
  
## <a name="embed-diagram-in-a-sharepoint-web-or-blog-page"></a>Внедрение схемы в страницу SharePoint, веб-сайта или блога
<a name="BM_7"> </a>

Вы можете использовать Visio для Интернета, чтобы вставить схему Visio (созданную в классическом приложении Visio) на странице SharePoint, веб-сайта или блога. Дополнительные сведения можно найти [в статье Visio for Web: вопросы](https://support.office.com/article/visio-online-frequently-asked-questions-e6647040-2fca-42ec-9fa5-d16a4e39e0ee)и ответы.
  
## <a name="find"></a>Поиск
<a name="BM_8"> </a>

В Visio для Интернета можно искать все вхождения слова, числа или фразы на странице, нажав кнопку с многоточием (...), а затем выбрав команду найти (или клавиши CTRL + F). Результаты появятся рядом со схемой. Visio выполняет поиск в именах, тексте и данных фигур, а также в определенных пользователем ячейках.
  
## <a name="font-formatting"></a>Форматирование шрифта
<a name="BM_9"> </a>

В Visio для Интернета невозможно изменить шрифт, размер шрифта и другие атрибуты оформления шрифта (такие как полужирный шрифт, курсив, Надстрочный знак или подчеркивание).
  
## <a name="hyperlinks"></a>Гиперссылки
<a name="BM_10"> </a>

В Visio для Интернета можно напрямую получить доступ к гиперссылкам, щелкнув фигуру.
  
## <a name="navigation"></a>Навигация
<a name="BM_11"> </a>

В Visio для Интернета можно переходить к другим страницам, щелкая вкладки в левом нижнем углу.
  
## <a name="office-add-ins"></a>Надстройки Office
<a name="BM_12"> </a>

Надстройки Office недоступны в Visio для Интернета.
  
## <a name="offline-viewing"></a>Просмотр в автономном режиме
<a name="BM_13"> </a>

Недоступно в Visio для Интернета. Для Office для Интернета требуется подключение к Интернету и Интернет-браузер. Для просмотра и редактирования схемы при отсутствии подключения к Интернету или локальному серверу Office Web Apps на компьютере должно быть установлено приложение Visio для настольных ПК. Если у вас установлено классическое приложение Visio, вы можете использовать все функции Visio для просмотра и редактирования схемы. При сохранении внесенные вами изменения добавляются в схему в Интернете. Вы также можете скачать и сохранить автономную копию схемы. Узнайте больше о [сервере Office Web Apps](https://technet.microsoft.com/library/ff431685.aspx).
  
## <a name="print-to-pdf"></a>Print to PDF
<a name="BM_14"> </a>

Печать в PDF недоступна в Visio для Интернета.
  
## <a name="proofing-tools"></a>Средства проверки правописания
<a name="BM_15"> </a>

Средства проверки правописания, такие как проверка правописания, перевод и пользовательские словари, недоступны в Visio для Интернета.
  
## <a name="real-time-co-authoring"></a>Real-time co-authoring
<a name="BM_16"> </a>

Недоступно в Visio для Интернета. Расширенные функции рецензента, например устранение конфликтов слияния и сравнение схем, доступны только в классическом приложении Visio. С помощью Visio для Интернета можно просматривать, добавлять и удалять комментарии.
  
## <a name="save-as-or-download-a-copy"></a>Save As or Download a copy
<a name="BM_17"> </a>

Visio для Интернета позволяет использовать параметр "Сохранить как" или "загрузить копию схемы", но для редактирования локальной копии необходимо иметь приложение Microsoft Visio для настольных ПК.
  
## <a name="share-a-diagram"></a>Общий доступ к схеме
<a name="BM_18"> </a>

Если схема сохранена в библиотеке документов SharePoint или в OneDrive для бизнеса, она уже находится в сети, поэтому к ней можно предоставить доступ, отправив по электронной почте не само вложение, а лишь ссылку. Пользователи с соответствующими разрешениями смогут просматривать ее в веб-браузере или на мобильном устройстве. Просто нажмите кнопку **Общий доступ** в правом верхнем углу. 
  
## <a name="smartart"></a>SmartArt
<a name="BM_19"> </a>

В Visio для Интернета невозможно добавить SmartArt в схему. Тем не менее, если элемент SmartArt был добавлен с помощью классического приложения Visio, он будет отображаться в Visio для Интернета.
  
## <a name="tell-me"></a>Помощник
<a name="BM_20"> </a>

Функция "рассказать мне" недоступна в Visio для Интернета. В классическом приложении Visio с помощью функции помощника можно быстро находить справку по нужным задачам.
  
## <a name="undo-and-redo"></a>Отмена и повтор
<a name="BM_21"> </a>

Функции отмены и возврата недоступны в Visio для Интернета.
  
## <a name="view-and-add-comments"></a>View and add comments
<a name="BM_22"> </a>

 В Visio для Интернета можно работать совместно с другими пользователями, просматривая, добавляя и отвечая на комментарии в схеме. 
  
## <a name="view-shape-data"></a>Просмотр данных фигур
<a name="BM_23"> </a>

Вы можете использовать Visio для Интернета, чтобы увидеть метаданные для схемы: просто щелкните фигуру, а затем щелкните **сведения о фигуре**.
  
## <a name="view-visio-files-in-the-browser"></a>Просмотр файлов Visio в браузере
<a name="BM_24"> </a>

Так как схема Visio сохраняется в Интернете, вы можете отправить ссылку на нее. После этого другие пользователи смогут просматривать ее в любом браузере. Им не потребуется классическое приложение Visio.
  
## <a name="zoom"></a>Масштабирование
<a name="BM_25"> </a>

Visio для Интернета включает ползунок, позволяющий увеличить и уменьшить масштаб. Кроме того, с помощью функции панорамного &amp; масштабирования можно увеличить область схемы.
