# <a name="visio-for-the-web"></a>Visio в Интернете

Visio для Интернета расширяет возможности Microsoft Visio в браузере, где вы можете создавать и редактировать схемы, сохраненные в облаке. Пользователи Microsoft Office 365 могут просматривать, печатать и совместно использовать схемы, а также добавлять комментарии в дороге. Если у вас есть подписка на Visio, вы можете использовать дополнительные функции, как описано в следующих разделах.
  
## <a name="apply-rich-formatting-to-text-and-shapes"></a>Применение форматирования к тексту и фигурам

Функции редактирования, такие как дополнительные цвета, градиенты, эффекты и стили, доступны в Visio для Интернета, а также в классическом приложении. Кроме того, можно добавить текстовое поле или выбрать из галереи фигур и применить стили, которые определяют эффекты заливки, контура и тени.
  
## <a name="arrange-objects"></a>Arrange objects

Вы можете перемещать, изменять размеры, поворачивать, переворачивать и упорядочивать фигуры и текстовые поля, а также отменять группировку фигур, чтобы работать с ними по отдельности в Visio для Интернета.
  
## <a name="build-mashup-solutions"></a>Создание гибридных решений

Приложение Visio для гибридного API веб-приложений JavaScript позволяет разработчикам получать доступ к веб-схеме Visio, ее страницам и ее фигурам, а также управлять ими. Дополнительные сведения можно найти [в статье Visio for Web: вопросы](https://support.office.com/article/e6647040-2fca-42ec-9fa5-d16a4e39e0ee)и ответы.
  
## <a name="create-diagrams"></a>Создание схем

Вы можете легко создать схемы в Visio для Интернета и сохранить их в SharePoint или OneDrive для бизнеса. Но вы можете создавать схемы в классическом приложении Visio, отправлять их в SharePoint или OneDrive для бизнеса, а затем делиться ими.
  
## <a name="cut-copy-and-paste"></a>Команды "Вырезать", "Копировать" и "Вставить"

Функции "вырезать", "Копировать" и "вставить" доступны в Visio для Интернета. Эти функции можно использовать для перемещения или удаления фигур и текста в схеме.
  
## <a name="edit-diagrams"></a>Редактирование схем

Visio для Интернета включает широкие возможности редактирования, которые можно использовать для внесения изменений в схему. Вы также можете изменить макет схемы с помощью различных цветов и макетов темы.
  
## <a name="embed-diagram-in-a-sharepoint-web-or-blog-page"></a>Внедрение схемы в страницу SharePoint, веб-сайта или блога

Вы можете использовать Visio для Интернета, чтобы вставить схему Visio (созданную в классическом приложении Visio) на странице SharePoint, веб-сайта или блога. Дополнительные сведения можно найти [в статье Visio for Web: вопросы](https://support.office.com/article/e6647040-2fca-42ec-9fa5-d16a4e39e0ee)и ответы.
  
## <a name="find"></a>Найти

В Visio для Интернета можно выполнить поиск каждого вхождения слова, числа или фразы на странице, нажав кнопку **Дополнительно** (многоточие), а затем выбрав **найти** (или нажав клавиши CTRL + F). Результаты появятся рядом со схемой. Visio выполняет поиск в именах, тексте и данных фигур, а также в определенных пользователем ячейках.
  
## <a name="font-formatting"></a>Форматирование шрифта

В Visio для Интернета можно изменить шрифт, размер шрифта и другие атрибуты оформления шрифта (например, полужирный шрифт, курсив, надстрочный или подчеркивание).
  
## <a name="hyperlinks"></a>Гиперссылки

В Visio для Интернета можно напрямую получить доступ к гиперссылкам, выбрав фигуру.
  
## <a name="navigation"></a>Навигация

В Visio для Интернета можно переходить к другим страницам, выбирая вкладки в левом нижнем углу.
  
## <a name="office-add-ins"></a>Надстройки Office

Надстройки Office пока недоступны в Visio для Интернета.
  
## <a name="offline-viewing"></a>Просмотр в автономном режиме

Недоступно в Visio для Интернета. Для Office для Интернета требуется подключение к Интернету и Интернет-браузер. Для просмотра и редактирования схемы при отсутствии подключения к Интернету или локальному серверу Office Web Apps на компьютере должно быть установлено приложение Visio для настольных ПК. Если на вашем компьютере установлено приложение Visio для настольных ПК, вы можете использовать все возможности Visio для просмотра и редактирования схемы в автономном режиме. При сохранении внесенные вами изменения добавляются в схему в Интернете. Вы также можете скачать и сохранить автономную копию схемы. Дополнительные сведения о сервере Office Web Apps можно найти в разделе [Office Web Apps Server](https://docs.microsoft.com/webappsserver/how-office-web-apps-work-on-premises-with-sharepoint-2013).
  
## <a name="print-to-pdf"></a>Print to PDF

Для веб-сайта можно выполнить печать в PDF или принтеры с помощью Visio.
  
## <a name="proofing-tools"></a>Средства проверки правописания

Средства проверки правописания, такие как проверка правописания, перевод и пользовательские словари, недоступны в Visio для Интернета. Однако доступны средства проверки читаемости и текстовые описания.
  
## <a name="real-time-co-authoring"></a>Real-time co-authoring

Пока недоступно в Visio для Интернета. С помощью Visio для Интернета можно просматривать, добавлять и удалять комментарии.
  
## <a name="save-as-or-download-a-copy"></a>Save As or Download a copy

Visio для веб-сайта позволяет использовать параметр "Сохранить как" или "загрузить копию схемы", но для редактирования локальной копии необходимо приложение Visio для настольных ПК.
  
## <a name="share-a-diagram"></a>Общий доступ к схеме

Если схема сохранена в библиотеке документов SharePoint или в OneDrive для бизнеса, она уже находится в сети, поэтому к ней можно предоставить доступ, отправив по электронной почте не само вложение, а лишь ссылку. Пользователи с соответствующими разрешениями смогут просматривать ее в веб-браузере или на мобильном устройстве. Просто выберите **Общая папка** в правом верхнем углу. Мы рекомендуем поделиться ссылкой, чтобы избежать создания дубликатов подключенных к данным схем.
  
## <a name="smartart"></a>SmartArt

В Visio для Интернета невозможно добавить SmartArt в схему. Тем не менее, если элемент SmartArt был добавлен с помощью классического приложения Visio, он будет отображаться в Visio для Интернета.
  
## <a name="tell-me"></a>Помощник

Функция "рассказать мне" недоступна в Visio для Интернета. В классическом приложении Visio с помощью функции помощника можно быстро находить справку по нужным задачам.
  
## <a name="undo-and-redo"></a>Отмена и повтор

Функции отмены и возврата доступны в Visio для Интернета.
  
## <a name="view-and-add-comments"></a>View and add comments

 В Visio для Интернета можно работать совместно с другими пользователями, просматривая, добавляя и отвечая на комментарии в схеме. 
  
## <a name="view-shape-data"></a>Просмотр данных фигур

Вы можете использовать Visio для Интернета, чтобы увидеть метаданные для схемы: просто выделите фигуру и выберите **сведения о фигуре**.
  
## <a name="view-visio-files-in-the-browser"></a>Просмотр файлов Visio в браузере

Так как схема Visio сохраняется в Интернете, вы можете отправить ссылку на нее. После этого другие пользователи смогут просматривать ее в любом браузере. Для них не требуется установленное приложение Visio для настольных компьютеров или лицензия Visio для просмотра схемы в браузере.
  
## <a name="zoom"></a>Масштабирование

Visio для Интернета включает ползунок, позволяющий увеличить и уменьшить масштаб. Кроме того, с помощью функции панорамного &amp; масштабирования можно увеличить область схемы.
