# <a name="visio-online"></a>Visio Online

Visio Online позволяет работать с Microsoft Visio в браузере и обрабатывать схемы прямо на веб-сайтах, на которых они размещены. Пользователи Microsoft Office 365 с Office Online могут просматривать схемы, делиться ими и вставлять комментарии, где бы они ни находились.
  
## <a name="apply-rich-formatting-to-text-and-shapes"></a>Применение форматирования к тексту и фигурам
<a name="BM_1"> </a>

Отсутствует в Visio Online. Функции редактирования, такие как дополнительные цвета, градиенты, эффекты и стили, доступны в классическом приложении Visio. С помощью Visio Online вы можете добавить текстовое поле, выбрать фигуру из галереи и применять стили, определяющие эффекты заливки, контуров и теней.
  
## <a name="arrange-objects"></a>Упорядочивание объектов
<a name="BM_2"> </a>

Эта функция пока что недоступна в Visio Online. В классическом приложении Visio вы можете перемещать, масштабировать, поворачивать и упорядочивать фигуры и текстовые поля, а также отменять группировку фигур, чтобы работать с ними по отдельности. 
  
## <a name="build-mashup-solutions"></a>Создание гибридных решений
<a name="BM_3"> </a>

API гибридных веб-приложений JavaScript для Visio Online позволяет разработчикам открывать веб-схемы Visio, их страницы и фигуры, а также управлять ими. Дополнительные сведения см. в статье [Visio в Интернете: вопросы и ответы](https://go.microsoft.com/fwlink/?linkid=825706).
  
## <a name="create-diagrams"></a>Создание схем
<a name="BM_4"> </a>

В Visio Online невозможно создавать схемы. Но вы можете создавать схемы в классическом приложении Visio, отправлять их в SharePoint или OneDrive для бизнеса, а затем делиться ими.
  
## <a name="cut-copy-and-paste"></a>Команды "Вырезать", "Копировать" и "Вставить"
<a name="BM_5"> </a>

Эта функция пока что недоступна в Visio Online. Вы можете перемещать и удалять фигуры и текст в схеме с помощью команд "Вырезать", "Копировать" и "Вставить" классического приложения Visio.
  
## <a name="edit-diagrams"></a>Редактирование схем
<a name="BM_6"> </a>

В Visio Online невозможно редактировать схемы. Чтобы внести изменения в схему, используйте классическое приложение Visio.
  
## <a name="embed-diagram-in-a-sharepoint-web-or-blog-page"></a>Внедрение схемы в страницу SharePoint, веб-сайта или блога
<a name="BM_7"> </a>

С помощью Visio Online вы можете вставить схему Visio (созданную в классическом приложении Visio) на страницу SharePoint, веб-сайта или блога. Дополнительные сведения см. в статье [Visio в Интернете: вопросы и ответы](https://go.microsoft.com/fwlink/?linkid=825706).
  
## <a name="find"></a>Поиск
<a name="BM_8"> </a>

В Visio Online можно найти все вхождения слова, числа или фразы на странице, нажав кнопку с многоточием (…) и выбрав команду "Найти" (либо нажав клавиши CTRL+F). Результаты появятся рядом со схемой. Visio выполняет поиск в именах, тексте и данных фигур, а также в определенных пользователем ячейках.
  
## <a name="font-formatting"></a>Форматирование шрифта
<a name="BM_9"> </a>

В Visio Online невозможно изменить шрифт, его размер и другие атрибуты внешнего вида (например, полужирный шрифт, курсив, надстрочный или подчеркнутый текст).
  
## <a name="hyperlinks"></a>Гиперссылки
<a name="BM_10"> </a>

В Visio Online можно напрямую переходить по гиперссылкам, щелкнув фигуру.
  
## <a name="navigation"></a>Навигация
<a name="BM_11"> </a>

В Visio Online можно переходить на другие страницы, выбирая вкладки в левой нижней части.
  
## <a name="office-add-ins"></a>Надстройки Office
<a name="BM_12"> </a>

Надстройки Office недоступны в Visio Online.
  
## <a name="offline-viewing"></a>Просмотр в автономном режиме
<a name="BM_13"> </a>

Возможность недоступна в Visio Online. Для использования Office Online необходимы подключение к Интернету и веб-браузер. Для просмотра и редактирования схем без подключения к Интернету и локальному серверу Office Web Apps на компьютере должно быть установлено классическое приложение Visio. Если у вас установлено классическое приложение Visio, вы можете использовать все функции Visio для просмотра и редактирования схемы. При сохранении внесенные вами изменения добавляются в схему в Интернете. Вы также можете скачать и сохранить автономную копию схемы. Дополнительные сведения см. в статье [Office Online Server](https://technet.microsoft.com/library/ff431685.aspx).
  
## <a name="print-to-pdf"></a>Функция "Сохранить в формате PDF"
<a name="BM_14"> </a>

Функция "Сохранить в формате PDF" недоступна в Visio Online.
  
## <a name="proofing-tools"></a>Средства проверки правописания
<a name="BM_15"> </a>

В Visio Online недоступны средства проверки правописания, такие как проверка орфографии и настраиваемые словари, а также возможности перевода.
  
## <a name="real-time-co-authoring"></a>Совместное редактирование в режиме реального времени
<a name="BM_16"> </a>

Отсутствует в Visio Online. Расширенные функции рецензента, например устранение конфликтов слияния и сравнение схем, доступны только в классическом приложении Visio. С помощью Visio Online можно просматривать, добавлять и удалять комментарии.
  
## <a name="save-as-or-download-a-copy"></a>Функции "Сохранить как" или "Загрузить копию"
<a name="BM_17"> </a>

В Visio Online можно использовать команду "Сохранить как" или скачать копию схемы, но для редактирования локальной копии необходимо классическое приложение Microsoft Visio.
  
## <a name="share-a-diagram"></a>Общий доступ к схеме
<a name="BM_18"> </a>

Если схема сохранена в библиотеке документов SharePoint или в OneDrive для бизнеса, она уже находится в сети, поэтому к ней можно предоставить доступ, отправив по электронной почте не само вложение, а лишь ссылку. Пользователи с соответствующими разрешениями смогут просматривать ее в веб-браузере или на мобильном устройстве. Просто нажмите кнопку **Общий доступ** в правом верхнем углу. 
  
## <a name="smartart"></a>SmartArt
<a name="BM_19"> </a>

С помощью Visio Online невозможно добавить объект SmartArt к схеме. Но если объект SmartArt был добавлен с помощью классического приложения Visio, он будет отображаться в Visio Online.
  
## <a name="tell-me"></a>Подробнее
<a name="BM_20"> </a>

Функция помощника недоступна в Visio Online. В классическом приложении Visio с помощью функции помощника можно быстро находить справку по нужным задачам.
  
## <a name="undo-and-redo"></a>Отмена и повтор
<a name="BM_21"> </a>

Функции отмены и повтора действий недоступны в Visio Online.
  
## <a name="view-and-add-comments"></a>Просмотр и добавление комментариев
<a name="BM_22"> </a>

 В Visio Online можно работать совместно с другими пользователями, просматривая, добавляя и отвечая на комментарии к схеме. 
  
## <a name="view-shape-data"></a>Просмотр данных фигур
<a name="BM_23"> </a>

С помощью Visio Online можно просматривать метаданные схемы. Просто нажмите фигуру и выберите **Сведения о фигуре**.
  
## <a name="view-visio-files-in-the-browser"></a>Просмотр файлов Visio в браузере
<a name="BM_24"> </a>

Так как схема Visio сохраняется в Интернете, вы можете отправить ссылку на нее. После этого другие пользователи смогут просматривать ее в любом браузере. Им не потребуется классическое приложение Visio.
  
## <a name="zoom"></a>Масштабирование
<a name="BM_25"> </a>

Visio Online включает в себя ползунок, позволяющий увеличивать и выход. Кроме того, можно использовать Pan &amp; функции масштабирования для увеличения области диаграммы.
