# <a name="microsoft-365-and-office-365-platform-service-description"></a>Описание служб платформы Microsoft 365 и Office 365

Microsoft 365 и Office 365 доступны в различных планах для максимального соответствия потребностям вашей организации. В этой статье описываются различия между планами, включая планы Exchange Online.
  
Microsoft 365 и Office 365 обеспечивают мощь облачных ресурсов для организаций любого размера, помогая экономить время и денежные ресурсы с действующие версиями. Планы Microsoft 365 и Office 365 сочетают знакомый набор Microsoft Office для настольных ПК с облачными версиями корпорации Майкрософт, коммуникации нового поколения и совместной работы (в том числе Microsoft Exchange Online, Microsoft SharePoint Online, Office в Интернете и Microsoft Skype для бизнеса Online), чтобы помочь пользователям работать почти через Интернет.
  
Сравнение функций в разных планах см. в [разделе "Эффективные инструменты](https://go.microsoft.com/fwlink/?LinkID=799177&amp;clcid=0x409) для поддержки предприятия" или более подробный список функций см. в описании соответствующей службы [в описаниях служб Microsoft 365 и Office 365.](../office-365-service-descriptions-technet-library.md)
  
- Если вас интересуют планы Office 365 для образования, начните со страницы [Office 365 для образования](office-365-education.md). 
    
- Если вас интересуют планы Office 365 US Government, начните со страницы [Office 365 для государственных организаций США](office-365-us-government/office-365-us-government.md). 
    
- Если вас интересует Office 365 под управлением 21Vianet, начните со страницы [Служба Office 365, которой управляет 21Vianet](office-365-operated-by-21vianet.md). 
    
- Если вас интересует Office 365 Germany, начните со страницы [Office 365 в Германии](office-365-germany.md). 
    
## <a name="office-365-plans"></a>Планы Office 365

Служба Office 365 доступна в разных планах для того, чтобы наилучшим образом удовлетворить потребности вашей организации. Сведения о различных планах, включая варианты автономных планов и сведения о переходе от одного плана к другому, см. в [разделах по планам Office 365.](office-365-plan-options.md)
  
Сведения о корпоративном лицензировании см. в статье ["Условия и документация о лицензировании".](https://go.microsoft.com/fwlink/?linkid=848696)
  
## <a name="microsoft-365-and-office-365-plans-and-pricing-for-nonprofits"></a>Планы и цены на некоммерческие организации Microsoft 365 и Office 365

Для организаций, которые соответствуют планам Microsoft 365 и Office 365 для некоммерческих организаций, эти планы включают те же функции, что и соответствующие бизнес-планы (например, Office 365 для некоммерческих организаций включает те же функции, что и Office 365 E5). Единственное отличие между эквивалентными планами — цена. Дополнительные сведения о планах Microsoft 365 и Office 365 для некоммерческих организаций см. в статье ["Сравнение предложений Microsoft 365 и Office 365 для некоммерческих](https://go.microsoft.com/fwlink/?LinkID=627221)организаций" и решений для некоммерческих [организаций.](https://go.microsoft.com/fwlink/?LinkID=786641)

## <a name="multi-geo-capabilities"></a>Поддержка нескольких регионов 

Поддержка нескольких регионов дает возможность отдельным организациям охватывать несколько географических центров обработки данных Office 365 и хранить их в выбранных регионах для Exchange Online и OneDrive для бизнеса. 
  
Доступные географические регионы: Азиатско-Тихоокеанский регион, Австралия, Канада, Европейский Союз, Индия, Япония, Корея, Соединенное Комиссия, Соединенные Штаты. 
  
Дополнительные сведения см. в [статье "Знакомство с поддержкой нескольких регионов" в Office 365.](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Introducing-Multi-Geo-in-Office-365/ba-p/107016)
