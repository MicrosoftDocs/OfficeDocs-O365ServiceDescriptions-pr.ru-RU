# <a name="microsoft-365-suite-features"></a>Компоненты набора Microsoft 365

Функции, перечисленные в этом разделе, — это набор веб-средств, которые помогают безопасно взаимодействовать и взаимодействовать на различных устройствах и в нескольких службах Office 365. Эти функции помогают налаживать новые связи в вашей организации, объединяя создание цифрового контента, хранилище и управление со средствами социальных сетей для поиска информации и обмена нею между пользователями.
  
## <a name="delve"></a>Delve

Delve позволяет пользователям искать и обнаруживать контент в Microsoft 365 на основе персонализированных данных. Delve — это первое решение на платформе Office Graph. Чтобы узнать больше о Delve, перейдите в ["Что такое Delve?".](https://go.microsoft.com/fwlink/?LinkId=511463)

## <a name="microsoft-bookings"></a>Microsoft Bookings

Microsoft Bookings — это интерактивное и мобильное приложение для предприятий, которые хотят предоставлять услуги клиентам на основе встреч. К примерам таких предприятий относятся кабинеты с оголевьями, санатории, санатории, юридические компании, поставщики финансовых услуг, консультанты, практикаторы в сфере здравоохранения, образовательные учреждения и автомагистрали. Служба Microsoft Bookings доступна клиентам в США и Канаде, у которых есть подписка на Microsoft 365 для бизнеса.
  
Дополнительные сведения о Microsoft Bookings можно найти в публикации [бизнес-календаря в Интернете с помощью Microsoft Bookings.](https://support.office.com/article/47403d64-a067-4754-9ae9-00157244c27d)

## <a name="microsoft-briefing-email"></a>Электронная почта microsoft Briefing

Сообщение электронной почты с кратким сообщением помогает пользователям использовать каждый день, помогая им:

* Подготовьтесь к собраниям, показывая связанные задачи или соответствующие документы.
* Управление задачами, идентифицированные по электронной почте, которые могут быть ожидающих.
* Уделяем основное внимание времени, резервировав календарь и отключая уведомления.

Электронная почта с кратким сообщением автоматически предоставляется всем пользователям Exchange Online. Дополнительные сведения см. в обзоре [электронной почты с кратким описанием.](https://docs.microsoft.com/Briefing/be-overview)

## <a name="microsoft-power-automate"></a>Microsoft Power Automate

Microsoft Power Automate позволяет автоматизировать процессы в различных приложениях. Flow позволяет подключать оповещения об электронной почте и мгновенных сообщениях, синхронизировать файлы между приложениями, копировать файлы из одной службы в другую, собирать данные из одного приложения и хранить их в другом и т. д. Для начала работы доступны шаблоны. Чтобы узнать больше о возможностях Flow и их использовании, перейдите на веб-сайт [Power Automate.](https://preview.flow.microsoft.com/)
  
## <a name="microsoft-forms"></a>Microsoft Forms

Microsoft Forms позволяет быстро и легко создавать пользовательские тесты, опросы, анкеты, регистрации и другие. После создания викторины или какой-либо формы можно предложить другим пользователям ответить на вопросы. При этом они могут использовать любой браузер, в том числе на мобильном устройстве. Данные форм, такие как результаты теста, можно легко экспортировать в Excel для дополнительного анализа или проверки. Чтобы узнать больше, [см. что такое Microsoft Forms?](https://support.office.com/forms).
  
## <a name="microsoft-graph-api"></a>API Microsoft Graph

С помощью API Microsoft Graph вы можете быстро создавать приложения, которые подключаются к множеством ресурсов, связей и аналитики через одну конечную https://graph.microsoft.com точку: Например, вы можете создавать пользовательские панели мониторинга, рабочий процесс и приложения, которые извлекать данные для вашей организации. Дополнительные сведения см. в [документации по Microsoft Graph.](https://go.microsoft.com/fwlink/?linkid=849595)
  
## <a name="microsoft-lists"></a>Microsoft Lists

Списки Майкрософт — это приложение Microsoft 365, которое помогает отслеживать информацию и упорядоизировать работу. Списки простые, интеллектуальные и гибкие, поэтому вы можете оставаться в верхней части того, что важнее всего для вашей команды. Отслеживайте проблемы, ресурсы, процедуры, контакты, инвентаризацию и другие функции с помощью настраиваемых представлений и интеллектуальных правил, чтобы все были оповещены и синхронизированы. С помощью готовых шаблонов вы можете быстро запускать списки в Интернете, в новом мобильном приложении или в Microsoft Teams. Так как это часть Microsoft 365, вы можете полагаться на корпоративные требования безопасности и соответствия требованиям.

Дополнительные сведения см. в указанных ниже статьях.

* [Общедоступный **Центр по внедрению списков Майкрософт**](https://aka.ms/ListsAdoption)

## <a name="microsoft-planner"></a>Планировщик (Майкрософт)

"Планировщик" (Майкрософт)  это инструмент, предоставляющий пользователям наглядный способ организации командной работы. Команды могут создавать планы, формировать и назначать задачи, делиться файлами, обсуждать ход выполнения работы, устанавливать сроки выполнения и обновлять статус. Кроме того, "Планировщик" позволяет связывать документы с конкретными задачами, совместно редактировать их и открывать беседы по задачам. Дополнительные сведения см. в статье [Знакомство с приложением "Планировщик" (Майкрософт)](https://go.microsoft.com/fwlink/?LinkID=718016&amp;clcid=0x4809). Это приложение пока не доступно в [Office 365 в Германии](office-365-germany.md).
  
## <a name="microsoft-power-apps"></a>Microsoft Power Apps

Microsoft Power Apps — это корпоративная служба, которая помогает быстро создавать, интегрировать и совместно использовать приложения, которые работают на любом устройстве. Power Apps позволяет использовать встроенные подключения или подключения, построенные вашей компанией, для подключения приложения к облачным и локальной службам, включая Microsoft 365 и Office 365, Dynamics CRM, OneDrive, SharePoint, SQL Server, базы данных Oracle и т. д. Вы можете легко поделиться своими приложениями Power Apps с коллегами, просто введя адрес электронной почты. Чтобы узнать больше, перейдите на веб-сайт Microsoft [Power Apps.](https://powerapps.microsoft.com/)
  
## <a name="microsoft-stream"></a>Microsoft Stream

Microsoft Stream — это корпоративная служба видео, которую сотрудники вашей организации могут использовать для безопасной отправки, просмотра и передачи видео. Вы можете поделиться записями занятий, собраний, презентаций, учебных семинаров и другими видеофайлами, которые будут полезны для совместной работы вашей команды. Microsoft Stream также позволяет легко делиться комментариями к видео и отмечать коды времени в комментариях и описаниях для ссылки на определенные моменты в видео.
  
## <a name="microsoft-sway"></a>Microsoft Sway

Microsoft Sway — это профессиональное приложение для разработки цифровых технологий для бизнеса, которое помогает вам и вашим коллегам выражать идеи с помощью интерактивного веб-холста. Встроенный механизм проектирования Microsoft Sway помогает создавать профессиональные, визуально привлекательные отчеты, презентации и другие функции без необходимости в форматирования или дополнительном обучении. Вы также можете изменить результаты, чтобы получить уникальный внешний вид и стиль, который вы хотите. Microsoft Sway делает ваше создание прекрасно смотреться в любом браузере на любом экране, и его можно использовать для совместного работы с коллегами и клиентами, просто отправив ссылку. Microsoft Sway помогает находить и совмещать все виды содержимого, не выходя из приложения, поэтому вы можете перетаскивать изображения, текст, видео и диаграммы прямо на холст. Дополнительные узнать см. в [Sway.](https://sway.com) Microsoft Sway пока не доступен в [Office 365 Germany.](office-365-germany.md)
  
## <a name="microsoft-365-apps-for-enterprise"></a>Приложения Microsoft 365 для предприятий

Приложения Microsoft 365 для предприятий — это всегда новый набор классических приложений, которые вы уже знаете (в том числе Word, PowerPoint, Excel, Outlook и Teams), доступных в качестве подписки. Приложения Microsoft 365 для предприятий включают монопольные интеллектуальные возможности, такие как идеи в [**Excel,**](https://go.microsoft.com/fwlink/p/?linkid=2109915)"Исследователи" в [**Word,**](https://go.microsoft.com/fwlink/p/?linkid=2109916)совместная работа в режиме реального [**времени,**](https://go.microsoft.com/fwlink/p/?linkid=2109917)Teams как центр командной работы и расширенные функции безопасности. В качестве пользовательской лицензии вы можете развернуть Office на пяти компьютерах с Windows или Mac, пяти планшетах и пяти мобильных устройствах. С помощью 1 ТБ облачного хранилища OneDrive вы можете получать доступ к файлам и совместно работать с ними из любого места.

## <a name="microsoft-365-groups"></a>Группы Microsoft 365

Группы в Microsoft 365 связывают пользователей с коллегами, сведениями и приложениями, которые им необходимы для совместной работы. Группы открыты по умолчанию для улучшения возможности обнаружения и общего доступа, но пользователи также могут создавать частные группы для конфиденциального контента. Дополнительные информацию о группах [см. в записях "Группы" в Microsoft 365.](https://support.office.com/Article/Find-help-about-Groups-in-Office-365-7a9b321f-b76a-4d53-b98b-a2b0b7946de1) Сведения об ограничениях для групп см. в под вопросе "Управление группами" в сведениях о группах [Microsoft 365.](https://go.microsoft.com/fwlink/?linkid=846714)

## <a name="microsoft-kaizala"></a>Microsoft Kaizala

Мобильное приложение Kaizala позволяет пользователям отправлять мгновенные сообщения и вложения, такие как изображения, документы, видео, аудиофайлы и другие, а также использовать карточки действий для опросов, опросов, заданий заданий, объявлений, учебных занятий и других. [Kaizala](https://products.office.com/en/business/microsoft-kaizala) работает на iPhone и Android. Чтобы узнать больше, перейдите на [веб-сайт Kaizala.](https://products.office.com/en/business/microsoft-kaizala)

## <a name="office-365-video"></a>Office 365 Видео

Office 365 Видео — это безопасное место для всей компании, в котором вы можете загружать, делиться и обнаруживать видео на своих устройствах на платформе microsoft Azure Media Services. Видео в Office 365 помогает привлечь внимание посетителей и проинформировать их, широко сообщить о идеях, а также сделать организации неуявным. Дополнительные узнать см. ## <a name="onenote-class-notebook"></a>Записная книжка OneNote для занятий

Записная книжка для занятий входит в состав OneNote и доступна для Office 365 для образования и Office 365 E5 для образования. Каждая записная книжка включает библиотеку содержимого для преподавателей для обмена контентом курса, пространство для совместной работы, в котором преподаватели и учащиеся могут работать вместе, и частную записную книжку для каждого учащегося. После создания записной книжки для занятий преподаватели и учащиеся могут использовать приложение OneNote для доступа к ней с любого устройства.

## <a name="to-do"></a>To Do

Microsoft To Do — это приложение для управления задачами, которое позволяет клиентам планировать свой день и управлять своими задачами (на работе и в жизни) с любого устройства.

С помощью Microsoft To Do клиенты могут:

* Синхронизация задач в Outlook, Teams, Планировщике и других
* Оставайтесь в центре внимания с помощью "Мой день", персонализированного ежедневного планировщика с предлагаемыми задачами
* Получите свои списки в любом месте, на любом устройстве или платформе
* Совместное выполнение списков и назначение задач коллегам, друзьям и членам семьи

Чтобы узнать больше, см. ["To Do" (На что нужно сделать)](https://to-do.microsoft.com/)

## <a name="whiteboard"></a>Доска

Доска (Майкрософт) — это свободный цифровой холст, в котором люди, содержимое и идеи объединяется. Вы можете использовать доску для совместной работы с командой для выполнения множества действий независимо от того, где находится ваша команда— в одном месте или в нескольких расположениях. Участники группы могут совместно работать на своих устройствах.

Дополнительные сведения о доске (Майкрософт) см. в приложении ["Доска" в Digital Online .](https://www.microsoft.com/microsoft-365/microsoft-whiteboard/digital-whiteboard-app)
