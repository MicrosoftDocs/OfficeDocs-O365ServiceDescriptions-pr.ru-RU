# <a name="office-365-suite-features"></a>Функции набора Office 365

Функции, перечисленные в этом разделе, представляют собой набор веб-средств, которые помогают обеспечить безопасность и безопасное взаимодействие на многих устройствах и нескольких службах Office 365. Эти функции помогают налаживать новые связи в вашей организации, объединяя создание цифрового контента, хранилище и управление со средствами социальных сетей для поиска информации и обмена нею между пользователями.
  
## <a name="microsoft-bookings"></a>Microsoft Резервирования
<a name="BKMK_Bookings"> </a>

Microsoft Bookings  это мобильное и веб-приложение для предприятий малого бизнеса, которые обслуживают клиентов по предварительной записи. К таким предприятиям относятся парикмахерские, стоматологические кабинеты, СПА, юридические фирмы, поставщики финансовых услуг, консультанты и автосервисы. Приложение Microsoft Bookings доступно для пользователей с подпиской на Office 365 бизнес премиум в США и Канаде. 
  
Дополнительные сведения о Microsoft Bookings см. в статье [Добро пожаловать в Microsoft Bookings](https://support.office.com/en-us/article/Say-hello-to-Microsoft-Bookings-47403d64-a067-4754-9ae9-00157244c27d?ui=en-US&amp;rs=en-US&amp;ad=US).
  
## <a name="microsoft-flow"></a>Microsoft Flow
<a name="BKMK_Flow"> </a>

Microsoft Flow позволяет автоматизировать рабочие процессы во всех приложениях. Flow позволяет подключать оповещения об электронной почте и мгновенных сообщениях, синхронизировать файлы между приложениями, копировать файлы из одной службы в другую, собирать данные из одного приложения и хранить их в другом и т. д. Для начала работы доступны шаблоны. Чтобы узнать больше о возможностях Flow, посетите веб-сайт [Microsoft Flow](https://flow.microsoft.com/en-us/).
  
## <a name="microsoft-forms"></a>Microsoft Forms
<a name="BKMK_Forms"> </a>

Microsoft Forms — это новая часть Office 365 для образовательных учреждений, которая позволяет быстро и легко создавать настраиваемые тесты, опросы, анкеты, регистрации и многое другое. После создания викторины или какой-либо формы можно предложить другим пользователям ответить на вопросы. При этом они могут использовать любой браузер, в том числе на мобильном устройстве. Данные форм, например результаты опросов, можно легко экспортировать в Excel для дополнительного анализа или ступенчатого анализа. Чтобы узнать больше, посмотрите, [что такое формы Microsoft Forms?](https://support.office.com/en-us/forms).
  
## <a name="microsoft-graph-api"></a>API Microsoft Graph
<a name="BKMK_Graph"> </a>

Используйте API Microsoft Graph, чтобы быстро создавать приложения, которые подключаются к обширным ресурсам, связям и аналитическим ресурсам через одну конечную https://graph.microsoft.comточку:. Например, вы можете создавать настраиваемые панели мониторинга, рабочие процессы и приложения, которые извлекают данные для вашей организации. Более подробную информацию можно найти в [документации по Microsoft Graph](https://go.microsoft.com/fwlink/?linkid=849595).
  
## <a name="microsoft-myanalytics"></a>Microsoft MyAnalytics
<a name="BKMK_MyAnalytics"> </a>

MyAnalytics предоставляет подробные сведения о двух ключевых факторах в вашей личной продуктивности: о том, как вы тратите время и кто тратится на работу. MyAnalytics просматривает повседневные рабочие работы, находит области, которые помогут вам получить от вас внимание, и предоставляет средства для более продуманного использования времени. Узнайте больше о [Microsoft MyAnalytics](https://products.office.com/business/myanalytics-personal-analytics) и [Microsoft MyAnalytics для администраторов Office 365](https://docs.microsoft.com/workplace-analytics/myanalytics/overview/mya-for-admins).
  
## <a name="microsoft-planner"></a>Планировщик (Майкрософт)
<a name="BKMK_Planner"> </a>

"Планировщик" (Майкрософт)  это инструмент, предоставляющий пользователям наглядный способ организации командной работы. Команды могут создавать планы, формировать и назначать задачи, делиться файлами, обсуждать ход выполнения работы, устанавливать сроки выполнения и обновлять статус. Кроме того, "Планировщик" позволяет связывать документы с конкретными задачами, совместно редактировать их и открывать беседы по задачам. Дополнительные сведения см. в статье [Знакомство с приложением "Планировщик" (Майкрософт)](http://go.microsoft.com/fwlink/?LinkID=718016&amp;clcid=0x4809). Это приложение пока не доступно в [Office 365 в Германии](office-365-germany.md).
  
## <a name="microsoft-powerapps"></a>Microsoft PowerApps
<a name="BKMK_PowerApps"> </a>

Microsoft PowerApps  это корпоративная служба, которая помогает быстро создавать, интегрировать и публиковать приложения, работающие на любом устройстве. PowerApps позволяет использовать встроенные и собственные механизмы подключения приложения к облачным и локальным службам, в том числе Office 365, Dynamics CRM, OneDrive, SharePoint, SQL Server, базам данных Oracle и т. д. Вы можете легко отправить свое приложение PowerApps коллегам, просто введя адрес электронной почты. Чтобы узнать больше, посетите веб-сайт [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/).
  
## <a name="microsoft-staffhub"></a>Microsoft StaffHub
<a name="BKMK_StaffHub"> </a>

Microsoft StaffHub  это облачное приложение, которое позволяет сотрудникам и менеджерам управлять расписаниями и поддерживать связь с помощью мобильных устройств. С помощью StaffHub сотрудники могут запрашивать отгулы, меняться сменами и общаться с коллегами. Сообщения можно отправлять друг другу или всей команде. Менеджеры могут в StaffHub отправлять сообщения, создавать расписания и отправлять новости команды или объявления компании. Чтобы узнать больше о StaffHub, посетите веб-сайт [Microsoft StaffHub](https://staffhub.office.com/) и [блог команды Office 365](https://blogs.office.com/2017/01/12/microsoft-staffhub-is-here/).
  
## <a name="microsoft-stream"></a>Microsoft Stream
<a name="BKMK_Stream"> </a>

Microsoft Stream — это корпоративная видеослужба, которую сотрудники Организации могут использовать для безопасной передачи, просмотра и совместного использования видео. Вы можете обмениваться записями классов, собраний, презентаций, учебных семинаров и других видеороликов, которые помогают совместно работать с группой. Кроме того, Microsoft Stream упрощает совместное использование комментариев в кодах времени для видео и маркеров в комментариях и описаниях, чтобы ссылаться на конкретные моменты в видеоролике.
  
## <a name="microsoft-sway"></a>Microsoft Sway
<a name="BKMK_Sway"> </a>

Microsoft Sway это профессиональное приложение Digital сторителлинг для бизнеса, помогающее вам и вашим коллегам выразить свои идеи с помощью интерактивного веб-холста. Встроенный модуль проектирования Microsoft Sway позволяет создавать профессионально оформленные отчеты, презентации и многое другое, не требуя расширенного форматирования или дополнительного обучения. Вы также можете изменить результаты, чтобы получить уникальный внешний вид и удобство. В Microsoft Sway внешний вид создан в любом браузере на любом экране, и к нему можно предоставить доступ коллегам и клиентам, просто отправив ссылку. Microsoft Sway помогает находить и объединять все виды контента, не выходя из приложения, поэтому вы можете перетаскивать изображения, текст, видео и диаграммы прямо на холст. Чтобы узнать больше, ознакомьтесь со статьей [Sway](https://sway.com). Microsoft Sway пока недоступен в [Office 365 Германии](office-365-germany.md).
  
## <a name="microsoft-teams"></a>Microsoft Teams
<a name="BKMK_Teams"> </a>

Microsoft Teams  это рабочее пространство в Office 365: на базе чата, которое помогает членам команд быстрее достигать целей. Microsoft Teams  это центр командной работы, который предоставляет мгновенный доступ к чатам, содержимому и инструментам Office 365:. В него встроены SharePoint и OneNote, а члены команд могут работать с документами Office непосредственно в приложении. Помимо чатов, Microsoft Teams также поддерживает видеозвонки и собрания, позволяя членам команд общаться в реальном времени как по запросу, так и по расписанию. Члены команды могут работать с несколькими командами и искать людей, чаты и файлы в любое время. Microsoft Teams работает на платформах Windows, Mac, Android, iOS, а также веб-платформах. Чтобы узнать больше, просмотрите [справку по Microsoft Teams](https://go.microsoft.com/fwlink/?linkid=834413). Рабочее пространство Microsoft Teams пока не доступно в [Office 365 в Германии](office-365-germany.md).
  
## <a name="office-delve"></a>Office Delve
<a name="BKMK_Delve"> </a>

Office Delve позволяет пользователям находить контент в Office 365: на основе персонализированной статистики. Delve  это первое решение на платформе Office Graph. Дополнительные сведения об Office Delve см. в статье [Что такое Office Delve](https://go.microsoft.com/fwlink/?LinkId=511463). 
  
## <a name="office-365-groups"></a>Группы Office 365
<a name="BKMK_Groups"> </a>

Группы в Office 365 помогают пользователям общаться с коллегами и получать доступ к нужным сведениям и приложениям, чтобы успешно сотрудничать. По умолчанию группы открыты, потому их легко находить и совместно использовать. Кроме того, пользователи могут создавать закрытые группы для конфиденциального содержимого. Дополнительные сведения о группах см. в статье [Группы в Office 365](https://support.office.com/en-US/Article/Find-help-about-Groups-in-Office-365-7a9b321f-b76a-4d53-b98b-a2b0b7946de1). Об ограничениях для групп читайте в разделе "Как можно управлять группами?" в статье [Сведения о группах в Office 365](https://go.microsoft.com/fwlink/?linkid=846714).
  
## <a name="office-365-video"></a>Office 365 Видео
<a name="BKMK_Video"> </a>

Office 365 Видео  это защищенный корпоративный ресурс для размещения, обмена и поиска видео на всех ваших устройствах на базе служб мультимедиа Microsoft Azure. Office 365 Видео поможет вам удержать зрителей, поделиться идеями с широкой аудиторией и обеспечить слаженную работу организации. Дополнительные сведения см. в статье [Знакомство с Office 365 Видео](https://blogs.office.com/2014/11/18/introducing-office-365-video/).
  
## <a name="onenote-class-notebook"></a>Записная книжка OneNote для занятий

Записная книжка для занятий является частью OneNote и доступна для Office 365 для образовательных учреждений и Office 365 в образовательных заведений. Каждая Записная книжка включает библиотеку контента для преподавателей, чтобы обмениваться контентом курса, пространством для совместной работы, в котором преподаватели и студенты могут совместно работать, а также частный портативный компьютер для каждого учащегося. После создания записной книжки для занятий преподаватели и студенты могут использовать приложение OneNote для доступа к нему с любого устройства.

## <a name="microsoft-kaizala"></a>Microsoft Kaizala

Microsoft Kaizala — это простое и безопасное приложение для управления мобильными работами, которое применяет уникальные возможности Microsoft 365, позволяя им подключаться и привлекать людей как к каталогу Организации, так и за его пределами, в том числе с контрактными работниками, поставщиками Партнеры, поставщики, клиенты и гражданские группы, использующие крупные и [гибкие типы групп](https://techcommunity.microsoft.com/t5/Microsoft-Kaizala-Blog/Model-your-communication-needs-using-Kaizala/ba-p/299710). Модель открытых каталогов Kaizala включает идентификацию на основе номеров телефонов для упрощения входящей миграции и простой пользовательский интерфейс для управления обменом сообщениями и работой. Мобильное приложение Kaizala позволяет пользователям отправлять мгновенные сообщения и вложения, такие как изображения, документы, видео, звук и многое другое, а также использовать карточки действий для опросов, опросов, назначений заданий, объявлений, обучение и многого другого. [Kaizala доступен в Office 365](https://products.office.com/en/business/microsoft-kaizala) и работает на iPhone и Android. Чтобы узнать больше, перейдите на [веб-сайт Kaizala](https://products.office.com/en/business/microsoft-kaizala).
