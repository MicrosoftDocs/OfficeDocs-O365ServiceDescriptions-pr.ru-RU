# <a name="office-365-suite-features"></a>Функции набора Office 365

Функциональные возможности, представленные в этом разделе, коллекция веб-средств, помогающих совместной работы и безопасный обмен данными между многих устройствах и несколько служб Office 365. Эти возможности помогают установить новую связь в пределах организации, объединяя цифрового содержимого, хранения и управление с помощью социальных средства обнаружения и обмен информацией между пользователями.
  
## <a name="microsoft-bookings"></a>Microsoft Резервирования
<a name="BKMK_Bookings"> </a>

Microsoft Bookings  это мобильное и веб-приложение для предприятий малого бизнеса, которые обслуживают клиентов по предварительной записи. К таким предприятиям относятся парикмахерские, стоматологические кабинеты, СПА, юридические фирмы, поставщики финансовых услуг, консультанты и автосервисы. Приложение Microsoft Bookings доступно для пользователей с подпиской на Office 365 бизнес премиум в США и Канаде. 
  
Дополнительные сведения о Microsoft Bookings см. в статье [Добро пожаловать в Microsoft Bookings](https://support.office.com/en-us/article/Say-hello-to-Microsoft-Bookings-47403d64-a067-4754-9ae9-00157244c27d?ui=en-US&amp;rs=en-US&amp;ad=US).
  
## <a name="microsoft-flow"></a>Microsoft Flow
<a name="BKMK_Flow"> </a>

Microsoft Flow позволяет автоматизировать рабочие процессы во всех приложениях. Flow позволяет подключать оповещения об электронной почте и мгновенных сообщениях, синхронизировать файлы между приложениями, копировать файлы из одной службы в другую, собирать данные из одного приложения и хранить их в другом и т. д. Для начала работы доступны шаблоны. Чтобы узнать больше о возможностях Flow, посетите веб-сайт [Microsoft Flow](https://flow.microsoft.com/en-us/).
  
## <a name="microsoft-forms"></a>Microsoft Forms
<a name="BKMK_Forms"> </a>

Microsoft Forms входит новые Education Office 365, который позволяет быстро и легко создавать настраиваемые викторины, опросы, анкеты, регистрации и др. При создании викторине или формы, можно пригласить других отвечать на него с помощью любого веб-браузера, даже на мобильных устройствах. Как отправлены результаты, встроенных аналитики могут использовать для оценки ответов. Данные формы, например результаты, можно легко экспортировать в Excel для дополнительного анализа и оценка успеваемости. Чтобы получить дополнительные сведения, обратитесь к разделу [возможности Microsoft Forms?](https://support.office.com/en-us/forms).
  
## <a name="microsoft-graph-api"></a>API Microsoft Graph
<a name="BKMK_Graph"> </a>

Использование Microsoft Graph API быстро разрабатывать приложения, которые подключаются к богатый набор ресурсов, связи и аналитика, все это одной конечной точки: https://graph.microsoft.com. Например можно создать настраиваемые панели мониторинга, рабочие процессы и приложениях, которые извлекают данные для вашей организации. Для получения дополнительных сведений обратитесь к [документации Microsoft Graph](https://go.microsoft.com/fwlink/?linkid=849595).
  
## <a name="microsoft-myanalytics"></a>Microsoft MyAnalytics
<a name="BKMK_MyAnalytics"> </a>

MyAnalytics предоставляет статистику, которая помогает оценить, насколько эффективно вы используете рабочее время. MyAnalytics анализирует вашу рабочую деятельность, находит области, которым следует уделять больше внимания, и предоставляет инструменты для более эффективного использования времени. Чтобы узнать больше о MyAnalytics, ознакомьтесь со статьями [Дополнительные сведения о работе со службой MyAnalytics](https://support.office.com/en-us/article/Learn-more-about-the-way-you-work-with-Microsoft-MyAnalytics-23462129-e512-40ee-acad-d968916c31b8?ui=en-US&amp;rs=en-US&amp;ad=US) и [Microsoft MyAnalytics для администраторов Office 365](https://support.office.com/en-us/article/Microsoft-MyAnalytics-for-Office-365-admins-77590915-6eb0-47a6-a72f-8116cfcfc2c7).
  
## <a name="microsoft-planner"></a>Планировщик Microsoft
<a name="BKMK_Planner"> </a>

"Планировщик" (Майкрософт)  это инструмент, предоставляющий пользователям наглядный способ организации командной работы. Команды могут создавать планы, формировать и назначать задачи, делиться файлами, обсуждать ход выполнения работы, устанавливать сроки выполнения и обновлять статус. Кроме того, "Планировщик" позволяет связывать документы с конкретными задачами, совместно редактировать их и открывать беседы по задачам. Дополнительные сведения см. в статье [Знакомство с приложением "Планировщик" (Майкрософт)](http://go.microsoft.com/fwlink/?LinkID=718016&amp;clcid=0x4809). Это приложение пока не доступно в [Office 365 в Германии](office-365-germany.md).
  
## <a name="microsoft-powerapps"></a>Microsoft PowerApps
<a name="BKMK_PowerApps"> </a>

Microsoft PowerApps  это корпоративная служба, которая помогает быстро создавать, интегрировать и публиковать приложения, работающие на любом устройстве. PowerApps позволяет использовать встроенные и собственные механизмы подключения приложения к облачным и локальным службам, в том числе Office 365, Dynamics CRM, OneDrive, SharePoint, SQL Server, базам данных Oracle и т. д. Вы можете легко отправить свое приложение PowerApps коллегам, просто введя адрес электронной почты. Чтобы узнать больше, посетите веб-сайт [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/).
  
## <a name="microsoft-staffhub"></a>Microsoft StaffHub
<a name="BKMK_StaffHub"> </a>

Microsoft StaffHub  это облачное приложение, которое позволяет сотрудникам и менеджерам управлять расписаниями и поддерживать связь с помощью мобильных устройств. С помощью StaffHub сотрудники могут запрашивать отгулы, меняться сменами и общаться с коллегами. Сообщения можно отправлять друг другу или всей команде. Менеджеры могут в StaffHub отправлять сообщения, создавать расписания и отправлять новости команды или объявления компании. Чтобы узнать больше о StaffHub, посетите веб-сайт [Microsoft StaffHub](https://staffhub.office.com/) и [блог команды Office 365](https://blogs.office.com/2017/01/12/microsoft-staffhub-is-here/).
  
## <a name="microsoft-stream"></a>Microsoft Stream
<a name="BKMK_Stream"> </a>

Поток Microsoft является службой видео enterprise, людей в организации можно использовать для отправки, просмотра и безопасное совместное использование видео. Можно совместно использовать записи классов, собрания, презентации, занятий и других видеозаписей, предназначенных для совместной работы группы. Поток Майкрософт также позволяет легко совместно использовать примечания на видео- и тег кодов времени в комментарии и описания для ссылки на конкретные точки в видео.
  
## <a name="microsoft-sway"></a>Microsoft Sway
<a name="BKMK_Sway"> </a>

Microsoft Sway является приложением профессиональный цифровой storytelling для бизнеса, которая поможет вам и ваших коллег express идеями с помощью интерактивного, веб-полотно. Модуль встроенных разработки Microsoft Sway поможет вам необходимо создать профессиональный, визуально привлекательных отчетов, презентаций и больше не требуется большой объем форматирования или Дополнительные учебные курсы. Можно также изменить результатов, чтобы получить уникальный внешний вид и функции, который будет. Microsoft Sway делает ваше создания прекрасно выглядят в любой браузер на любом экране, а можно использовать совместно с коллегами и клиентами, просто отправив ссылку. Microsoft Sway помогает, поиск и извлекать все типы контента, не выходя из приложения, поэтому вы можно перетаскивать изображения, текст, видео и диаграмм непосредственно в вашей полотно. Чтобы получить дополнительные сведения, обратитесь к разделу [Sway](https://sway.com). Microsoft Sway еще не доступен в [Office 365 Германии](office-365-germany.md).
  
## <a name="microsoft-teams"></a>Microsoft Teams
<a name="BKMK_Teams"> </a>

Microsoft Teams  это рабочее пространство в Office 365: на базе чата, которое помогает членам команд быстрее достигать целей. Microsoft Teams  это центр командной работы, который предоставляет мгновенный доступ к чатам, содержимому и инструментам Office 365:. В него встроены SharePoint и OneNote, а члены команд могут работать с документами Office непосредственно в приложении. Помимо чатов, Microsoft Teams также поддерживает видеозвонки и собрания, позволяя членам команд общаться в реальном времени как по запросу, так и по расписанию. Члены команды могут работать с несколькими командами и искать людей, чаты и файлы в любое время. Microsoft Teams работает на платформах Windows, Mac, Android, iOS, а также веб-платформах. Чтобы узнать больше, просмотрите [справку по Microsoft Teams](https://go.microsoft.com/fwlink/?linkid=834413). Рабочее пространство Microsoft Teams пока не доступно в [Office 365 в Германии](office-365-germany.md).

По умолчанию группы открыты, потому их легко находить и совместно использовать. Кроме того, пользователи могут создавать закрытые группы для конфиденциального содержимого.
