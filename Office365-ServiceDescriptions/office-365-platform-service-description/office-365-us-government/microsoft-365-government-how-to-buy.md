## <a name="microsoft-365-for-us-government-eligibility-and-validation"></a>Microsoft 365 для государственных организаций и проверки соответствия требованиям США

Microsoft 365 для государственных учреждений (США) предоставляется подходящим клиентам государственных учреждений и организациям, которые не являются государственными для хранения или обработки контролируемых данных. Критерии приемлемости для этих двух категорий клиентов согласованы в облачных органах Майкрософт.

Для всех клиентов, которые подключаются к облаку Microsoft для государственных организаций, доступны оба варианта: GCC и GCC, а среда DoD — эксклюзивное использование отдела обороны США. Доступность и цена службы отличаются, а GCC остается главный имиджевый баннерным движением для всех клиентов, которые не содержат неклассифицированных данных FedRAMP High или DoD (CUI).

Подходящего клиента для государственных учреждений определяется одним из следующих вариантов:

- Федеральный орган, определенный как бюро, Office, агентство, отдел или другой объект правительства США
- Состояние/локальная сущность, которая определяется одним из следующих вариантов:
    1. Любой орган штата или местных учреждений США
    2. Любой район США, бораугх, коммонвеалс, город, округ, населенный элемент, товншип, округ для специального назначения или другой подобный тип правительственных санкций, который находился законами штата клиента и находится в юрисдикции клиента стате'с и географические границы
    3. Округ Колумбия, Коммонвеалс Пуэрто-Рико, Гуам, Американское Самоа, Виргинские острова США и Северные Марианские острова
- Объект Трибал, определенный как Федеративная признанный объект Трибал, пригодный для финансирования и служб из отдела США в соответствии с его состоянием в виде индийского трибе, или, в случае внедрения, собственного собственного регионального набора услуг виллаже или Аляска

Неправительственные организации, которые содержат следующие типы данных и могут предоставить указанные доказательства для процесса проверки в облаке Microsoft правительством:

Коммерческая частная сущность с данными, которые подчиняются нормативным требованиям. Допустимые типы правительственных данных: 
- Международный трафик в подлокотниках (ITAR)
- Контролируемая Неклассифицированная информация (CUI)
- Неклассифицированные контролируемые сведения о подразделении (DoD) (УКНИ)
- Отдел энергии (Петров) УКНИ
- Информация о уголовной юстиции (КЖИ)
- Сведения о взаимодействии уровня обороны
- Другие типы данных, которым требуется Office 365 для государственных организаций

Международный коммерческий объект может быть определен, хотя подходящие данные могут быть необходимы для приобретения службы через свой филиал США.

Для получения доступа к правительственным решениям Microsoft 365 для государственных учреждений потребуется подтверждение участия в одной из указанных выше групп.
