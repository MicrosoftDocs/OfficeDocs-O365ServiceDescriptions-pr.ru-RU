## <a name="microsoft-365-for-us-government-eligibility-and-validation"></a>Microsoft 365 для государственных организаций допустимости США и проверки подлинности
<a name="BM2-customer"> </a>

Microsoft 365 для государственных учреждений США правительственные клиентов и не государственных организаций Спонсируемые хранения и обработки управляемой сведения. В облаке государственных организаций Microsoft согласуется критериям для этих двух категорий клиентов.

GCC и высокой GCC и услуги, доступны любому пользователю, который подходит для государственных организаций Microsoft Cloud и средой министерства обороны США является использовать только министерства обороны США. Доступность службы и цены отличаются и GCC остается движения герой для всех клиентов, в которых не содержатся высоким FedRAMP или министерства обороны США контролируемых Неклассифицированные сведения (CUI).

Подходящих правительственных клиентов определяется с помощью одного следующее:

-   Федеральное ведомство, определен как бюро, office, бюро, отдела или другой сущности правительство США
-   Объект state/local определена как один из следующих действий:
    1. Учреждения состояния или местное правительственное в США.
    
    2. Любой Район США, Округ, Содружества, Город, государственный орган, Город, township, Округ особого назначения или других того же типа государственных instrumentality в законов состояние клиента и находится в состоянии клиента инспекции и географическое границы
    3. Округ Колумбия, Содружество Пуэрто-Рико, Гуам, Американское Самоа, Виргинские острова Соединенных Штатах Америки и Северный Марианские о-ва
-   Семейные сущности, определен как федеральными распознанный семейные сущности, допустимость финансирование и служб из отдела внутренней США в поскольку они его состояние, как Индии глубинах или Аляска, собственная центре или Аляска региональных собственный Корпорация

Не государственных организаций, которые содержат следующие типы данных и может предоставить перечисленных экспериментов оплачиваются процесс проверки Microsoft Cloud государственных организаций:

Средство профессиональной закрытый сущности с данными, зависит от правила. Следующие типы данных обслуживаемых государственных организаций: 
- Международные трафика в руки (соответствующих международным правилам ТОРГОВЛИ)
- Управляет Неклассифицированные сведения (CUI)
- Отдел из обороны США Неклассифицированные управляются ядерного сведения (UCNI)
- Отдел электроэнергии (Петров) UCNI
- Сведения о преступник юстиции (CJI)
- Министерства обороны влияние уровня данных
- Другие типы данных, которые требуют государственных 365 Microsoft

Международные коммерческие сущности может определять, хотя регулируемого данных может потребоваться приобрести службы с помощью их дочерним подразделением США.

Требуется для доступа к Microsoft 365 государственных и услуги указать обоснования членство в одну из групп, перечисленных выше.
