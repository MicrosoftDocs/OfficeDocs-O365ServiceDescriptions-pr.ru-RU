## <a name="whats-microsoft-365-government"></a>Что такое Microsoft 365 для государственных органов?

Microsoft 365 для государственных организаций — это набор возможностей облачного программного обеспечения для повышения производительности, безопасности и мобильности, разработанный для государственных учреждений США и подрядчиков, спонсируемых для удержания контролируемой, неклассифицированной информации. В уникальных средах, которые соответствуют самым строгим требованиям соответствия требованиям, Microsoft 365 для государственных организаций — это облачное предложение для государственных организаций США, которое максимально соответствует функциям и возможностям коммерческих облачных корпоративных предложений Майкрософт.

## <a name="microsoft-365-government-eligibility-and-validation"></a>Соответствие требованиям и проверка Microsoft 365 для государственных органов

Microsoft 365 для государственных организаций доступна для соответствующих требованиям государственных организаций и неправительственных организаций, спонсируемых для удержания или обработки контролируемой информации. Критерии соответствия требованиям для этих двух категорий клиентов согласованы в облаке Microsoft Government Cloud.

Предложения GCC и GCC High доступны любому клиенту, который имеет право на использование Microsoft Government Cloud, а среда DoD доступна исключительно для министерства обороны США. Доступность службы и цены отличаются, и GCC остается самыми высокими предложениями для всех клиентов, которые не удерживают управляемые неклассифицированные сведения (CUI) FedRAMP High или DoD.

Подходящий клиент для государственных органов определяется одним из следующих ок.

- Федеральное агентство, определенное как агентство, офис, агентство, отдел или другой объект правительства США
- Объект state/local, определенный как один из следующих:
    - Любое агентство штата или местного правительства в США.
    - Любые округи США, загон, территории, город, загон, город, округ, округ специального назначения или другой аналогичный тип правительственных полномочий, установленных законами штата клиента и расположенных в пределах юрисдикции и географических границ клиента
    - Округ Колумбия, Пуэрто-Рико, Гуам, Американское Самоа, Виргинские острова США и Северные Марианские острова
- Заметивая сущность, определяемая как распознавая на уровне федерального правительства сущность, подходящая для получения финансов и услуг из министерства внутренних дел США, которая имеет статус "посвеялий", или, в Штате, собственной территории или внутренней региональной корпорации.

Неправительственные организации, которые имеют следующие типы данных и могут предоставить перечисленные доказательства, имеют право на проверку Microsoft Government Cloud:

Коммерческое частное предприятие с данными, подавными нормативным актам. К принятым типам данных для государственных органов относятся:
- Международный трафик в оружейных организациях (ITAR)
- Контролируемые неклассифицированные сведения (CUI)
- Отдел обороны (DoD) Неклассифицированные управляемые сведения о UCNI (UCNI)
- Отдел по энергетической безопасности (DoE) UCNI
- Сведения о злоумышленниках (CJI)
- Данные об уровне влияния на защиту министерства обороны
- Другие типы данных, которые требуют Microsoft 365 для государственных служб

Международное коммерческое предприятие может иметь право, хотя для покупки службы через его дочернее подразделение в США могут потребоваться регулируемые данные.

Для доступа к предложениям Microsoft 365 для государственных служб потребуется подтверждение членства в одной из перечисленных выше групп.
