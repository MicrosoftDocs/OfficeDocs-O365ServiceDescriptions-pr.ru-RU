## <a name="whats-microsoft-365-government"></a>Что такое Office 365 для государственных организаций?

Microsoft 365 правительственные учреждения — это набор возможностей программного обеспечения для облачной работы, безопасности и мобильности, адаптированный для государственных учреждений США и подрядчиков, для хранения контролируемой неклассифицированной информации. Доставляется с помощью уникальных сред, удовлетворяющих максимально строгому числу требований соответствия требованиям, Microsoft 365 правительственные учреждения — это облачное предприятие для государственных организаций США, которое соответствует таким возможностям, как возможности и возможности коммерческого предприятия корпорации Майкрософт.

## <a name="microsoft-365-government-eligibility-and-validation"></a>Проверка правильности и проверка соответствия требованиям для государственных учреждений Microsoft 365

Microsoft 365 правительственные учреждения доступны для пользователей и организаций, не являющихся правительственными учреждениями, которые могут хранить или обрабатывать контролируемую информацию. Критерии приемлемости для этих двух категорий клиентов согласованы в облачных органах Майкрософт.

Для всех клиентов, которые подключаются к облаку Microsoft для государственных организаций, доступны оба варианта: GCC и GCC, а среда DoD — эксклюзивное использование отдела обороны США. Доступность и цена службы отличаются, а GCC остается предложением главный Имиджевый баннер для всех пользователей, которые не содержат неклассифицированных данных FedRAMP High или DoD (CUI).

Подходящего клиента для государственных учреждений определяется одним из следующих вариантов:

- Федеральный орган, определенный как бюро, Office, агентство, отдел или другой объект правительства США
- Состояние/локальная сущность, которая определяется одним из следующих вариантов:
    - Любой орган штата или местных учреждений США
    - Любой район США, бораугх, коммонвеалс, город, округ, населенный элемент, товншип, округ для специального назначения или другой подобный тип правительственных клиентов, который находился законами штата и в пределах клиента, а также в территориальной области клиента.
    - Округ Колумбия, Коммонвеалс Пуэрто-Рико, Гуам, Американское Самоа, Виргинские острова США и Северные Марианские острова
- Объект Трибал, определенный как Федеративная признанный объект Трибал, пригодный для финансирования и служб из отдела США в соответствии с его состоянием в виде индийского трибе, или, в случае внедрения, собственного собственного регионального набора услуг виллаже или Аляска

Неправительственные организации, которые содержат следующие типы данных и могут предоставить указанные доказательства для процесса проверки в облаке Microsoft правительством:

Коммерческая частная сущность с данными, которые являются предметом нормативных требований. Допустимые типы правительственных данных:
- Международный трафик в подлокотниках (ITAR)
- Контролируемая Неклассифицированная информация (CUI)
- Неклассифицированные контролируемые сведения о подразделении (DoD) (УКНИ)
- Отдел энергии (Петров) УКНИ
- Информация о уголовной юстиции (КЖИ)
- Сведения о взаимодействии уровня обороны
- Другие типы данных, которым требуется Office 365 для государственных организаций

Международный коммерческий объект может быть определен, хотя подходящие данные могут быть необходимы для приобретения службы через свой филиал США.

Для получения доступа к правительственным решениям Microsoft 365 для государственных учреждений потребуется подтверждение участия в одной из указанных выше групп.
