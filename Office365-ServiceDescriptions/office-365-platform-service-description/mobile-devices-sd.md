# <a name="mobile-devices-sd"></a>Мобильные устройства [SD]

Многие мобильные телефоны, планшетные ПК и других мобильных устройств можно использовать с помощью Microsoft Office 365. 
  
- Сведения касательно конкретных мобильных устройств см. в статье [Сравнение работы Office 365 на различных мобильных устройствах](https://go.microsoft.com/fwlink/p/?LinkId=282337).
    
- Информацию об использовании электронной почты на мобильных устройствах см. в описании службы [Клиенты и мобильные устройства](../exchange-online-service-description/clients-and-mobile-devices.md). 
    
- Сведения о функции "Управление мобильными устройствами для Office 365" см. в статье [Обзор управления мобильными устройствами (MDM) для Office 365](https://go.microsoft.com/fwlink/?linkid=808602).
    
## <a name="feature-availability"></a>Доступность функций

Сведения о доступности функций в планах Office 365 см. в статье [Описание служб платформы Office 365](https://technet.microsoft.com/en-us/library/office-365-platform-service-description.aspx).
