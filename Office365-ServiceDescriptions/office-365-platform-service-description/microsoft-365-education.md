# <a name="microsoft-365-education"></a>Microsoft 365 для образования

Microsoft 365 доступен в различных планах, чтобы наилучшим образом удовлетворить потребности вашей организации. В этой статье приводится сравнение планов Microsoft 365 и Office 365 для образования с описанием компонентов, включенных в каждый из них.
  
Microsoft 365 представляет собой полноценную систему, включающую Office 365, Windows 10 и Enterprise Mobility and Security. В приведенной ниже таблице перечислены компоненты планов Office 365 для образования A1, A3 и A5, а также соответствующие компоненты Microsoft 365 для образования A3 и A5. Сравнение функций в планах для бизнеса и предприятий см. в статье "Сравнение планов [Microsoft 365](https://go.microsoft.com/fwlink/?linkid=799177)для бизнеса" или в более подробном списке функций см. соответствующее описание службы в описаниях [служб Microsoft 365 и Office 365.](../office-365-service-descriptions-technet-library.md) Чтобы найти статьи и сведения о поддержке, см. [справку по &amp; Office.](https://support.office.com/)
