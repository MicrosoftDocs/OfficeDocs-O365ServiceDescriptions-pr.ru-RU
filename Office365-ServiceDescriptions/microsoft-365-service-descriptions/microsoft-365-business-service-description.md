# <a name="microsoft-365-business-premium-service-description"></a>Описание службы Microsoft 365 бизнес премиум

Microsoft 365 бизнес премиум — это интегрированное решение, которое объединяет лучшую в своем классе производительность Microsoft 365 с расширенными службами безопасности и управления устройствами, чтобы защитить ваш бизнес от сложных угроз. В этом описании службы рассматриваются варианты и возможности планов. Дополнительные сведения об Office 365 см. в описаниях служб [Microsoft 365 и Office 365.](../office-365-service-descriptions-technet-library.md)

### <a name="notes"></a>Примечания
<sup>1</sup> Включает Exchange Online (план 1).<br/>
<sup>2</sup> Azure Information Protection (AIP) включает поддерживаемые функции управления правами на данные (IRM). AIP (план 1) входит в состав Microsoft 365 бизнес премиум. <br/>
<sup>3</sup> Microsoft 365 бизнес премиум включает архивация на базе Exchange Online подписку. Каждый подписчик на архивацию на базе Exchange Online изначально получает 100 ГБ в архивном почтовом ящике. Если включен автоматически расширяющийся архив, то по достижении объема в 100 ГБ автоматически добавляется место. Дополнительные сведения [см. в обзоре неограниченного архива.](https://docs.microsoft.com/office365/securitycompliance/unlimited-archiving) <br/>
<sup>4.</sup> Доступно в [https://securescore.office.com](https://securescore.office.com) . Требуются разрешения администратора. Дополнительные сведения см. в [оценке безопасности (Майкрософт).](https://docs.microsoft.com/office365/securitycompliance/microsoft-secure-score)<br/>
<sup>5.</sup> Упрощенное управление устройствами с Windows 10, встроенное в Центр администрирования Microsoft 365. Возможности управления Intune для iOS, Android, macOS и других кроссплатформенных средств управления устройствами встроены в Центр администрирования управления устройствами, также лицензированные для использования с Microsoft 365 бизнес премиум. Сторонними приложениями и конфигурацией таких вещей, как профили Wi-Fi и сертификаты VPN, также можно управлять в Центре администрирования управления устройствами. <br/>
<sup>6</sup> Этот план входит в состав Microsoft Stream (план 2), который включает видеоконференции, транскрибцию речи в текст, глубокий поиск и обнаружение лиц.<br/>
<sup>Microsoft</sup> 365 поставляется вместе с Microsoft Defender для Office 365 (план 1). Дополнительные сведения см. в сведениях о функции безопасности и соответствия требованиям [Microsoft 365](https://docs.microsoft.com/microsoft-365/business/security-features)бизнес премиум.<br/>
<sup>8</sup> подписчиков Microsoft 365 бизнес премиум имеют лицензию на использование всех возможностей Intune для iOS, Android, MacOS и других средств управления кроссплатформенными устройствами. Функции, недоступные в Центре администрирования Microsoft 365 бизнес премиум, например управление сторонними приложениями и настройка профилей WiFi, сертификатов VPN и т. д., можно управлять в полной консоли Intune.</br>
<sup>9</sup> Если у вас есть устройства с лицензиями на Windows 7 и 8.1 Профессиональная, Microsoft 365 бизнес премиум предоставляет обновление до Windows 10 Pro. Однако эта лицензия не охватывает Windows 7.1 или 8 Домашняя с Windows 10 Pro. С помощью Microsoft 365 бизнес премиум вы не только получаете функции безопасности Microsoft Defender, которые включены в [Windows Pro,](https://wfbdevicemanagementprod.blob.core.windows.net/windowsforbusiness/Windows10_CommercialEdition_Comparison.pdf)вы также получаете доступ к дополнительным функциям, таким как защита сети от веб-угроз и правила предотвращения вторжений на хост-серверы для снижения рисков программ-вымогателей. При настройке учетной записи Microsoft 365 бизнес премиум вы выбираете параметры защиты приложений и конфигурации устройств. Эти параметры определяют, какие политики автоматически применяются к устройствам при их присоединении к Azure Active Directory.</br>
