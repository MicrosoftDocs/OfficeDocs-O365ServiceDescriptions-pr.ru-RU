# <a name="microsoft-dynamics-365-online-service-description"></a>Описание веб-службы Microsoft Dynamics 365

С декабря 2016 года функциональность Dynamics CRM входит в состав Dynamics 365 (набора интеллектуальных бизнес-приложений). Dynamics 365 объединяет в себе функции управления отношениями с клиентами (CRM) и планирования ресурсов предприятия (ERP), предоставляя новые приложения для управления бизнес-функциями.  
  
Вот эти приложения:
  
- Dynamics 365 for Sales
    
- Dynamics 365 for Customer Service
    
- Dynamics 365 for Field Service
    
- Dynamics 365 for Project Service Automation
    
- Dynamics 365 for Operations
    
- Dynamics 365 Financials
    
Дополнительные сведения о функциях Dynamics 365 см. на сайте [Microsoft Dynamics 365](https://www.microsoft.com/en-us/dynamics365/home). 
  
## <a name="dynamics-365-plans"></a>Планы Dynamics 365

Dynamics 365 представлен в двух версиях: для предприятий и для бизнеса. В этих двух версиях доступно несколько планов, разработанных специально для крупных и мелких предприятий. Приложения также доступны по отдельности. 
  
Дополнительные сведения о функциях Dynamics 365 см. на сайте [Microsoft Dynamics 365](https://www.microsoft.com/en-us/dynamics365/home).
