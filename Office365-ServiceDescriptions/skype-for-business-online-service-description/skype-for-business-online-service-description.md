# <a name="skype-for-business-online-service-description"></a>Описание службы Skype для бизнеса online

> [!NOTE]
> Группами Майкрософт теперь — основной клиент для обмена сообщениями, собраний и вызов в Office 365. Запуск 1 октября 2018, новых клиентов Office 365 на 500 рабочих мест или меньше будет onboarded для групп Майкрософт и не смогут получить доступ к Скайп для бизнеса в Интернет. Клиенты, которые уже используют Скайп для бизнеса в Интернет сможете продолжить это Да (включая подготовки новых пользователей) до их завершения их перехода на группами Майкрософт.

> [!NOTE]
> Скайп для бизнеса Online план 1 вывода из продажи по состоянию на 10 октября. Это означает, что будет больше не смогут получить Скайп для бизнеса Online план 1, для обмена мгновенными сообщениями, присутствие и Peer-to-peer голосовой и видеосвязи. Эти функции будут поддерживаться в все предложения Office 365, включая группы, в том числе Office 365 F1. По истечении плана в конце текущего терминов для имеют постоянный доступ к этих функций необходимо хранить переключитесь в другой план. Мы рекомендуем оценке план Office 365 F1 для пользователей, которые в настоящий момент назначены Скайп для бизнеса Online план 1 назначен. 

 **Хотите знать, какие лицензии, вам необходимо купить для получения определенного Скайп для функций бизнес? В разделе [Скайп для бизнеса Общие сведения о лицензировании](https://go.microsoft.com/fwlink/?LinkID=799524).**
  
Skype для бизнеса Online  это служба мгновенных сообщений для проведения собраний и разговоров из практически любого места в любое время. Эта служба предоставляет пользователям доступ к сведениям о присутствии, обеспечивает обмен мгновенными сообщениями, аудио- и видеозвонки, полнофункциональные собрания по сети и широкие возможности проведения веб-конференций. 
  
Служба Skype для бизнеса размещена на мультитенантных серверах, которые поддерживают нескольких клиентов одновременно. Эти серверы размещены в центрах обработки данных Майкрософт и доступны с ряда устройств пользователей через корпоративную сеть или Интернет.
