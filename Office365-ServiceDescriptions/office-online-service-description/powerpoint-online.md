# <a name="powerpoint-for-the-web"></a>PowerPoint в Интернете

PowerPoint для Интернета (ранее PowerPoint Web App) расширяет возможности Microsoft PowerPoint в веб-браузере, где вы можете работать с презентациями непосредственно на веб-сайте, где хранится презентация. Пользователи Microsoft 365 с Office в Интернете могут просматривать, создавать и редактировать файлы в дороге.
  
Редактор PowerPoint — это интерфейсный веб-компонент, который создает поверхность редактирования на основе браузера, которая позволяет пользователям работать с документами без потери точности.
  
Если нажать кнопку **Открыть в PowerPoint** на панели инструментов веб-сайта PowerPoint, презентация откроется в классическом приложении PowerPoint (если на компьютере установлено приложение Microsoft PowerPoint 2010 или более поздней версии).
  
Узнайте, как [скачать и установить Office с помощью Microsoft 365 для бизнеса на компьютере](https://support.office.com/en-us/article/Download-and-install-Office-using-Office-365-for-business-on-a-PC-72977511-dfd1-4d8b-856f-405cfb76839c).
  
## <a name="alignment-bullets-numbers"></a>Alignment, bullets, numbers

В PowerPoint для Интернета можно изменить выравнивание абзаца, применить маркеры или нумерацию, а также изменить уровень маркированного или нумерованного текста. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="apply-basic-transitions-and-animations"></a>Применение основных переходов и анимаций

Выбирайте нужные элементы в коллекции эффектов перехода и анимации. PowerPoint для веб-сайта поддерживает 8 переходов и анимацию 37. Дополнительные эффекты анимации и переходы, не поддерживаемые в PowerPoint для веб-сайта, сохраняются в презентации и отображаются в слайд-шоу, но их нельзя изменить в PowerPoint для Интернета. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="apply-rich-formatting-to-text-shapes-and-pictures"></a>Apply rich formatting to text, shapes, and pictures

Недоступно в PowerPoint для Интернета. Расширенные функции форматирования, например дополнительные цвета, градиенты, пипетка, эффекты и стили, доступны в классическом приложении PowerPoint. С помощью PowerPoint для Интернета можно добавить текстовое поле или выбрать из галереи фигур и применить стили, которые определяют эффекты заливки, контура и тени. 
  
## <a name="apply-themes-and-theme-variants"></a>Применение тем и их вариантов

Выбирайте нужные элементы из галереи встроенных тем и их вариантов при добавлении нового слайда в презентацию. Темы, не поддерживаемые в PowerPoint для веб-сайта, сохраняются в презентации, но возможность изменять темы недоступна в PowerPoint для Интернета. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="office-add-ins"></a>Надстройки Office

PowerPoint для веб-сайта поддерживает только контентные надстройки для Office. Они интегрируют веб-функции как контент, который можно показать для презентации. Узнайте больше о [типах надстроек для Office](https://go.microsoft.com/fwlink/p/?LinkId=282310).
  
## <a name="arrange-objects"></a>Arrange objects

Перемещение, изменение размера, поворот и поупорядочивание фигур и текстовых полей в слоях в PowerPoint для веб-части. Разгруппируйте фигуры, чтобы работать с каждой из них по отдельности. Узнайте больше о [различиях, касающихся использования презентаций в браузере и классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="broadcast-slide-show"></a>Broadcast slide show

Для широковещательного показа слайдов на удаленную аудиторию через PowerPoint в Интернете требуется классическое приложение PowerPoint. С помощью PowerPoint пользователи могут просматривать презентации в Интернете, независимо от того, установлено ли для них классическое приложение PowerPoint. Узнайте больше о [широковещательных слайд-шоу](https://go.microsoft.com/fwlink/?LinkId=272765).
  
## <a name="clipboard"></a>Буфер обмена

С помощью PowerPoint для Интернета можно вырезать, копировать и вставлять содержимое в презентацию. Пользователь может копировать и вставлять текст между Office для веб-приложений, а также между Office для веб-приложений и настольных приложений Microsoft Office на компьютере. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="create-and-manage-slides"></a>Create and manage slides

Добавление, изменение порядка, дублирование, скрытие и удаление слайдов в PowerPoint для веб-сайта. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="create-custom-animation"></a>Create custom animation

PowerPoint для Интернета включает коллекцию эффектов анимации. Анимации, не поддерживаемые в PowerPoint для веб-сайта, сохраняются в презентации и отображаются в слайд-шоу, но их нельзя изменить в PowerPoint для Интернета. Для создания пользовательских анимаций необходимо установить на компьютере классическое приложение PowerPoint. Узнайте больше о [создании пользовательских анимаций в классическом приложении PowerPoint 2013](https://go.microsoft.com/fwlink/p/?LinkId=282348).
  
## <a name="design-tools-advanced"></a>Средства оформления, расширенные

Недоступно в PowerPoint для Интернета. Расширенные инструменты дизайна, например создание образцов слайдов и изменение макетов, доступны только в классическом приложении PowerPoint. В PowerPoint для Интернета можно использовать собственный файл шаблона в качестве основы для создания новых файлов или выбора предопределенного списка тем.
  
## <a name="dropbox"></a>Dropbox

Dropbox — служба размещения файлов с облачным хранилищем, синхронизацией файлов, личным облаком и клиентским программным обеспечением.
  
## <a name="embed-presentation-on-web-or-blog-page"></a>Внедрение презентации в веб-страницу или блог

С помощью PowerPoint в Интернете вы можете внедрять презентации на веб-сайтах или в блогах, чтобы любой пользователь мог просматривать свои сведения даже в том случае, если у них нет классического приложения Microsoft PowerPoint. Встроенные средства просмотра воспроизводят анимацию, переходы, звук и видео так же, как и в режиме просмотра с полной четкостью. Узнайте больше о [внедрении презентаций в веб-страницу или блог](https://go.microsoft.com/fwlink/?LinkId=272766).
  
## <a name="font-formatting"></a>Font formatting

PowerPoint для Интернета позволяет применять полужирный шрифт, курсив, подчеркивание, шрифт, размер и цвет текста. Можно также использовать форматирование по образцу для копирования формата всей фигуры.
  
## <a name="full-fidelity-reading-view"></a>Режим чтения с высокой четкостью

Просмотр презентаций в том виде, в&mdash;котором они предназначены для отображения анимаций, переходов, аудио-и видеороликов.
  
## <a name="full-ink-support"></a>Full ink support

Рукописный ввод невозможно вставить в PowerPoint для Интернета, но в PowerPoint для веб-сайта отображаются ожидаемые элементы.
  
## <a name="full-selection-of-animations-and-transitions"></a>Full selection of animations and transitions

PowerPoint для веб-сайта поддерживает 8 переходов и анимацию 37. Более полный набор анимаций и переходов доступен в классическом приложении PowerPoint. 
  
## <a name="headers-and-footers"></a>Верхние и нижние колонтитулы

Верхние и нижние колонтитулы, включая даты и номера слайдов, не могут быть вставлены, изменены или удалены в PowerPoint для Интернета, но PowerPoint для веб-страниц отображается должным образом. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="hyperlinks"></a>Гиперссылки

Вставка, редактирование и переход по гиперссылкам. Ссылки на закладки работают, но их нельзя редактировать в PowerPoint для Интернета.
  
## <a name="insert-online-video"></a>Вставка видео из Интернета

Видео- и аудиосодержимое воспроизводится в режиме чтения или слайд-шоу (предельный размер файла — 100 МБ). Видео из Интернета можно вставлять из YouTube, а элементы управления мультимедиа можно изменять, перемещать и удалять в PowerPoint для Интернета. Но для вставки видео- и аудиосодержимого с других ресурсов потребуется классическое приложение Microsoft PowerPoint.
  
## <a name="integration-with-excel-for-charts"></a>Интеграция с Excel для вставки диаграмм

Недоступно в PowerPoint для Интернета. С помощью PowerPoint для Интернета можно просматривать диаграммы Excel в существующей презентации, но вы не можете редактировать или вставлять диаграмму Excel в презентацию с помощью PowerPoint для Интернета. Узнайте больше о [копировании диаграмм Excel в PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272773).
  
## <a name="navigation---slide-sorter"></a>Навигация — Сортировщик слайдов

Режим сортировщика слайдов, который предоставляет представление слайдов в виде эскизов, упрощает сортировку и упорядочивание слайдов. С помощью режима сортировщика слайдов можно упорядочивать слайды, добавлять разделы и сортировать слайды по разным категориям.
  
## <a name="offline-viewing-and-authoring"></a>Просмотр и редактирование в автономном режиме

PowerPoint для Интернета запускается из веб-браузера и использует подключение к Интернету. Чтобы иметь доступ к презентациям в автономном режиме, необходимо установить на компьютер классическое приложение Microsoft PowerPoint, которое можно использовать для просмотра и редактирования слайдов.
  
## <a name="picture-cropping"></a>Обрезка рисунков

Улучшайте кадрирование изображений с помощью инструмента обрезки. Просто щелкните любой маркер обрезки на краю изображения и перетаскивайте его, пока не получите желаемый результат.
  
## <a name="pictures"></a>Изображения

С помощью PowerPoint для Интернета можно вставлять рисунки, сохраненные на компьютере, или вставлять изображения из изображений Bing. Изображения можно перемещать и обрезать, а также менять их размер и применять к ним множество стилей. Более сложные функции для работы с изображениями, такие как применение эффектов, недоступны в PowerPoint для Интернета. Вы не можете создавать снимки экрана в PowerPoint для Интернета, но снимки экрана в презентации отображаются как изображения в PowerPoint для Интернета. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="present-online-through-skype-for-business-or-the-office-presentation-service"></a>Present online through Skype for Business or the Office Presentation Service

Недоступно в PowerPoint для Интернета. Классическое приложение Microsoft PowerPoint позволяет создавать презентации с помощью службы презентаций Office, бесплатной общедоступной службы, которая позволяет другим пользователям отслеживать их в веб-браузере. Узнайте больше о [службе презентаций Office](https://go.microsoft.com/fwlink/?LinkId=272774).
  
## <a name="presenter-view"></a>Presenter view

Недоступно в PowerPoint для Интернета. Только классическое приложение Microsoft PowerPoint позволяет докладчику управлять ходом презентации и использовать заметки, аннотации и средства масштабирования. PowerPoint для Интернета не поддерживает эти функции.
  
## <a name="print-to-pdf"></a>Print to PDF

С помощью PowerPoint в Интернете вы можете напечатать презентацию в средстве чтения PDF-файлов, где все макеты и форматирование слайдов будут напечатаны должным образом. Узнайте больше о [базовых задачах, которые можно выполнить с помощью PowerPoint для Интернета](https://go.microsoft.com/fwlink/?LinkId=272775).
  
## <a name="proofing-tools"></a>Средства проверки правописания

Вы можете проверить орфографию и задать язык проверки правописания, используя встроенный словарь PowerPoint для Интернета. Но в PowerPoint для Интернета не используется пользовательский словарь, не включающий перевод или тезаурус.
  
## <a name="real-time-co-authoring"></a>Real-time co-authoring

В PowerPoint можно работать одновременно с несколькими авторами в PowerPoint для Интернета и PowerPoint 2016. Сведения о присутствии в режиме реального времени помогают понять, где совместное редактирование работает в презентации, чтобы не создавать конфликты при внесении изменений. PowerPoint 2013 поддерживает одновременное редактирование, но отсутствует признак присутствия, и изменения не видны сразу нескольким авторам, пока документ не будет сохранен. Дополнительные сведения о совместном редактировании в режиме реального времени в PowerPoint можно посмотреть в статье [Совместная работа над презентациями PowerPoint](https://go.microsoft.com/fwlink/?linkid=844014).
  
## <a name="reviewer-tools-advanced"></a>Средства рецензентов, расширенные

Недоступно в PowerPoint для Интернета. Расширенные функции рецензента, например конфликты слияния и сравнение презентаций, доступны только в классическом приложении PowerPoint. С помощью PowerPoint для Интернета можно просматривать, добавлять, редактировать и удалять комментарии.
  
## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>Управление правами. Применение и использование службы управлением правами на доступ к данным и защиты паролем

Недоступно в PowerPoint для Интернета. В PowerPoint для Интернета отображаются презентации, защищенные с помощью управления правами на доступ к данным (IRM). Тем не менее, эти презентации невозможно редактировать в браузере, и вы не можете создавать презентации, защищенные с помощью IRM, в PowerPoint для Интернета. PowerPoint в Интернете не может открывать презентации, которые подписаны цифровой подписью или зашифрованы с помощью пароля. 
  
## <a name="run-slide-show"></a>Run slide show

С помощью PowerPoint для Интернета вы можете запустить презентацию с помощью веб-браузера и подключения к Интернету. Узнайте больше о [базовых задачах, которые можно выполнить с помощью PowerPoint для Интернета](https://go.microsoft.com/fwlink/?LinkId=272775).
  
## <a name="save-as-or-download-a-copy"></a>Сохранение или Загрузка копии

PowerPoint для Интернета сохраняет свою работу каждый раз при внесении изменений. Команда " **сохранить** " недоступна. Скачать копию можно, но чтобы ее отредактировать на компьютере, нужно классическое приложение Microsoft PowerPoint.
  
## <a name="shapes"></a>Фигуры

В PowerPoint для Интернета невозможно вставить объекты Word Art и диаграммы, но в PowerPoint для веб-страниц они отображаются должным образом. Редактирование и форматирование текста в режиме редактирования. Если вы хотите применить текстовые эффекты к Word Art, вам потребуется классическое приложение Microsoft PowerPoint.
  
## <a name="share"></a>Share

Если презентация сохранена в библиотеке документов SharePoint, она уже находится в сети, и вы можете предоставить к ней общий доступ, просто отправив ссылку на нее вместо вложения в сообщение электронной почты. Пользователи с соответствующими разрешениями могут просматривать ее в веб-браузере или на мобильном устройстве. Узнайте больше об [общем доступе к презентации](https://go.microsoft.com/fwlink/?LinkId=272776).
  
## <a name="slide-show"></a>Slide show

PowerPoint для веба воспроизводит слайд-шоу в полноэкранном режиме. Для перелистывания слайдов вперед нажмите ПРОБЕЛ. Анимации слайдов отображаются, но между слайдами поддерживаются только переходы "Исчезание" и "Стирание". Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="smartart"></a>SmartArt

Вы можете вставить SmartArt, переключиться на другую структуру или цветовую схему, применять стили SmartArt и редактировать текст с помощью PowerPoint для Интернета. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="table-creation-editing-and-formatting"></a>Создание, редактирование и форматирование таблиц

Таблицы можно создавать и редактировать в PowerPoint для Интернета, а PowerPoint для Интернета поддерживает большинство табличных функций. Для расширенных функций, таких как объединение и разбивка ячеек, используйте классическое приложение PowerPoint. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="tell-me"></a>Помощник

Если вам нужно выполнить какие-либо действия в PowerPoint для Интернета, но вы не знаете, как это сделать, вы можете использовать функцию поиска, чтобы быстро найти то, что вы искали. Эта функция распознает задачи, которые вы пытаетесь выполнить, и помогает сделать это быстрее, предоставляя варианты.
  
## <a name="undo-and-redo"></a>Отмена и повтор

Отмена (CTRL + Z) и повтор (CTRL + Y) неопределенного количества последних действий в течение текущего сеанса редактирования в активной презентации. Журнал отмены действий сбрасывается, если время сеанса редактирования истекает или пользователь переключается в режим чтения больше чем на 30 секунд. Дополнительные сведения о [сочетаниях клавиш в PowerPoint для Интернета](https://go.microsoft.com/fwlink/?LinkId=272950).
  
## <a name="view-and-add-comments"></a>View and add comments

Если файлы хранятся в SharePoint Online или OneDrive для бизнеса, вы можете добавлять, изменять или удалять комментарии. 
  
Если файлы хранятся в OneDrive, вы можете добавлять, изменять или удалять комментарии в режиме редактирования, а также просматривать и обновлять комментарии в режиме чтения.
  
## <a name="view-and-edit-slide-notes"></a>View and edit slide notes

С помощью PowerPoint в Интернете заметки для каждого слайда можно отобразить или скрыть. Также можно добавить заметки в режиме редактирования.
  
## <a name="wysiwyg-viewing"></a>Просмотр в режиме WYSIWYG

Редактируйте презентацию практически в том же виде, что и при печати или показе готового продукта.
