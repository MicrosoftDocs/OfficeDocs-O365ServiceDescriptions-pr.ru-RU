# <a name="powerpoint-for-the-web"></a>PowerPoint для Интернета

PowerPoint для Интернета (ранее PowerPoint Web App) расширяет возможности Microsoft PowerPoint в веб-браузере, где вы можете работать с презентациями непосредственно на веб-сайте, где хранится презентация. Пользователи Microsoft Office 365 с Office в Интернете могут просматривать, создавать и редактировать файлы в дороге.
  
Редактор PowerPoint  это интерфейсный веб-компонент, предоставляющий возможность выполнять редактирование в веб-браузере, что позволяет работать над документами без потери точности.
  
Если нажать кнопку **Открыть в PowerPoint** на панели инструментов веб-сайта PowerPoint, презентация откроется в классическом приложении PowerPoint (если на компьютере установлено приложение Microsoft PowerPoint 2010 или более поздней версии). 
  
Узнайте, как [с помощью Office 365 для бизнеса скачать набор Office и установить его на компьютере](https://support.office.com/en-us/article/Download-and-install-Office-using-Office-365-for-business-on-a-PC-72977511-dfd1-4d8b-856f-405cfb76839c).
  
## <a name="alignment-bullets-numbers"></a>Alignment, bullets, numbers
<a name="bkmk_AlignmentBulletsNumbers"> </a>

В PowerPoint для Интернета можно изменить выравнивание абзаца, применить маркеры или нумерацию, а также изменить уровень маркированного или нумерованного текста. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="apply-basic-transitions-and-animations"></a>Apply basic transitions and animations
<a name="bkmk_Applybasictransitions"> </a>

Выбирайте нужные элементы в коллекции эффектов перехода и анимации. PowerPoint для веб-сайта поддерживает 8 переходов и анимацию 37. Дополнительные эффекты анимации и переходы, не поддерживаемые в PowerPoint для веб-сайта, сохраняются в презентации и отображаются в слайд-шоу, но их нельзя изменить в PowerPoint для Интернета. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).
  
## <a name="apply-rich-formatting-to-text-shapes-and-pictures"></a>Apply rich formatting to text, shapes, and pictures
<a name="bkmk_Applyrichformatting"> </a>

Вы не можете создавать снимки экрана в PowerPoint для Интернета, но снимки экрана в презентации отображаются как изображения в PowerPoint для Интернета. Узнайте больше о [различиях в использовании презентаций в браузере и в классическом приложении PowerPoint](https://go.microsoft.com/fwlink/?LinkId=272763).

## <a name="real-time-co-authoring"></a>Real-time co-authoring
<a name="bkmk_Coauthoring"> </a>

В PowerPoint можно работать одновременно с несколькими авторами в PowerPoint для Интернета и PowerPoint 2016. Сведения о присутствии в режиме реального времени помогают понять, где совместное редактирование работает в презентации, чтобы не создавать конфликты при внесении изменений. PowerPoint 2013 поддерживает одновременное редактирование, но отсутствует признак присутствия, и изменения не видны сразу нескольким авторам, пока документ не будет сохранен. Дополнительные сведения о совместном редактировании в режиме реального времени в PowerPoint можно посмотреть в статье [Совместная работа над презентациями PowerPoint](https://go.microsoft.com/fwlink/?linkid=844014).

<a name="bkmk_AdvancedReviewerTools"> </a>

## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>Управление правами. Применение и использование службы управлением правами на доступ к данным и защиты паролем
<a name="bkmk_CreationPasswords"> </a>

Недоступно в PowerPoint для Интернета. В PowerPoint для Интернета отображаются презентации, защищенные с помощью управления правами на доступ к данным (IRM). Тем не менее, эти презентации невозможно редактировать в браузере, и вы не можете создавать презентации, защищенные с помощью IRM, в PowerPoint для Интернета. PowerPoint в Интернете не может открывать презентации, которые подписаны цифровой подписью или зашифрованы с помощью пароля.

Редактирование и форматирование текста в режиме редактирования. Чтобы применить текстовый эффект к объекту Word Art, потребуется классическое приложение Microsoft PowerPoint.
  
## <a name="share"></a>Общий доступ
<a name="bkmk_Share"> </a>

Пользователи с соответствующими разрешениями могут просматривать ее в веб-браузере или на мобильном устройстве. Узнайте больше об [общем доступе к презентации](https://go.microsoft.com/fwlink/?LinkId=272776).
  
## <a name="slide-show"></a>Slide show
<a name="bkmk_Slideshow"> </a>

PowerPoint для веба воспроизводит слайд-шоу в полноэкранном режиме. Для перелистывания слайдов вперед нажмите ПРОБЕЛ. Анимации слайдов отображаются, но между слайдами поддерживаются только переходы "Исчезание" и "Стирание". ## <a name="tell-me"></a>Помощник
<a name="Bkmk_TellMe_PPT"> </a>

Если вам нужно выполнить какие-либо действия в PowerPoint для Интернета, но вы не знаете, как это сделать, вы можете использовать функцию поиска, чтобы быстро найти то, что вы искали. Эта функция распознает задачи, которые вы пытаетесь выполнить, и помогает сделать это быстрее, предоставляя варианты.
  
## <a name="undo-and-redo"></a>Undo and redo
<a name="bkmk_Undoredo"> </a>

Отмена (CTRL + Z) и повтор (CTRL + Y) неопределенного количества последних действий в течение текущего сеанса редактирования в активной презентации. Журнал отмены действий сбрасывается, если время сеанса редактирования истекает или пользователь переключается в режим чтения больше чем на 30 секунд. Дополнительные сведения о [сочетаниях клавиш в PowerPoint для Интернета](https://go.microsoft.com/fwlink/?LinkId=272950).
