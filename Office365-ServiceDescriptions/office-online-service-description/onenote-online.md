# <a name="onenote-for-the-web"></a>OneNote для Интернета

Используйте OneNote в Интернете (ранее OneNote Web App), чтобы делать заметки в Интернете в записной книжке OneNote, которую вы можете добавить из любого места и легко делиться с другими пользователями. Все пользователи Office 365 могут просматривать и легко редактировать файлы Office с помощью Office для Интернета. 
  
## <a name="advanced-collaboration-new-content-shows-as-unread-presence"></a>Расширенные возможности совместной работы. Отображение нового содержимого как непрочитанного; сведения о присутствии
<a name="bkmk_AdvancedCollaboration"> </a>

Недоступно в OneNote для Интернета. В классическом приложении OneNote можно легко определить и проверить подлинность других авторов с помощью интегрированных профилей. Поиск в записной книжке изменений, редакций по именам авторов и просмотр всех недавних изменений при возвращении в общую записную книжку.
  
## <a name="apply-tags"></a>Применение тегов
<a name="bkmk_Tags"> </a>

С помощью OneNote в Интернете вы можете применять различные теги к заметкам для упрощения организации и дальнейших действий. Например, можно пометить заметки как вопросы, задачи или контактные данные. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
  
## <a name="office-add-ins"></a>Надстройки Office
<a name="bkmk_AppsforOffice"> </a>

OneNote для Интернета поддерживает только контентные надстройки для Office. Контентные надстройки интегрируют веб-функции как содержимое, которое может быть показано в документе. Дополнительные сведения о [типах надстроек для Office](https://go.microsoft.com/fwlink/p/?LinkId=282310).
  
## <a name="audio-notes-recording"></a>Запись звуковых заметок
<a name="bkmk_AudioVideoLinked"> </a>

Недоступно в OneNote для Интернета. В классическом приложении OneNote можно записывать аудио- и видеозаметки, напрямую связанные с любыми текстовыми заметками, которые вы делаете во время записи. OneNote для Интернета не воспроизводит аудио-и видеоконтенты, но они сохраняются в записной книжке, и вы можете скачать звуковые и видеофайлы на компьютер для их воспроизведения. 
  
## <a name="bullets-and-numbering"></a>Маркеры и нумерация
<a name="bkmk_BulletsNumbering"> </a>

В OneNote для Интернета можно выбрать три стиля маркеров или пять стилей нумерации. Используйте кнопки увеличить отступ и уменьшить отступ для изменения уровня списка существующих маркированных и нумерованных списков в записной книжке, а также созданных в OneNote для Интернета. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).

Страницы и разделы можно изменять, форматировать и перемещать или копировать между записными книжками. Узнайте больше об [использовании OneNote для веб-записной книжки](https://go.microsoft.com/fwlink/p/?LinkId=272945).

<a name="bkmk_DownloadCopyEmbeddedOfficeFiles"> </a>

Классическое приложение OneNote 2013 добавляет возможность внедрять электронные таблицы Excel и рисунки Visio в заметки. Узнайте больше о [внедрении файлов Microsoft Office в OneNote](https://go.microsoft.com/fwlink/p/?LinkId=271843).

## <a name="edit-embedded-files"></a>Редактирование внедренных файлов
<a name="EmbeddedFileEdit"> </a>

С помощью OneNote для Интернета вы можете скачать встроенные файлы на компьютер, чтобы открыть их. В классическом приложении OneNote можно вкладывать практически любой компьютерный файл в любую часть заметок, что сохранит копию файла в записной книжке. В OneNote можно даже вставлять детали собраний и задач Outlook, а также отправлять электронным сообщением страницу OneNote в Outlook. Можно также создавать или импортировать электронные таблицы Excel и схемы Visio прямо в OneNote и редактировать их информацию непосредственно в заметках. Вставленные файлы показываются в виде значков на странице заметок. Дважды щелкните по любому значку, чтобы открыть файл.

## <a name="ink-viewing"></a>Просмотр рукописных фрагментов
<a name="bkmk_InkViewing"> </a>

В OneDrive OneNote для веба отображается рукописный ввод, но формулы отображаются в виде заполнителей. В SharePoint можно просматривать рукописный ввод, формулы отображаются в виде заполнителей, а фигуры скрыты. Они не могут быть вставлены или изменены в OneNote для Интернета. В OneDrive можно выбирать и удалять фигуры, данные рукописного ввода и формулы-заполнители. В SharePoint можно выбирать и удалять заполнители для этих объектов.
  
## <a name="inking-ink-to-text-and-math-customizable-pens-drawing-tools"></a>Рукописный ввод: рукописное сообщение в виде текста и математических символов, настраиваемые ручки, средства рисования
<a name="bkmk_Inking"> </a>

Недоступно в OneNote для Интернета. В классическом приложении OneNote можно плавно рисовать, очищать и редактировать пальцем, пером или мышью. Если вы предпочитаете писать, а не набирать текст, OneNote может преобразовывать рукописный текст. Если вы откроете записную книжку из библиотеки документов SharePoint, все формулы и данные рукописного ввода в записной книжке будут отображаться в виде заполнителей.

## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>Управление правами: Применение и использование службы управлением правами на доступ к данным и защиты паролем
<a name="bkmk_RightsManagement"> </a>

OneNote для Интернета отображает записные книжки, защищенные службой управления правами на доступ к данным (IRM). Однако эти записные книжки невозможно редактировать в браузере, и вы не можете создавать записные книжки, защищенные с помощью IRM, в OneNote для Интернета. OneNote для Интернета не может открывать документы, зашифрованные с помощью пароля. Расширенные функции защиты документов, такие как создание записных книжек с защитой IRM и применение защиты паролем, доступны только в классическом приложении OneNote.

Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).

Недоступно в OneNote для Интернета. В классическом приложении OneNote можно использовать шаблон как оформление страницы, которое можно применить к новым страницам в записной книжке, чтобы они имели привлекательный фон, единообразный внешний вид или согласованный макет.
  
## <a name="undo-and-redo"></a>Отмена и повтор
<a name="bkmk_UndoRedo"> </a>

С помощью клавиатуры можно отменить (Ctrl + Z) или повторить (Alt + F7) последние действия для каждой страницы, которая редактируется в активной записной книжке в OneNote для веб-сайта. OneNote для Интернета предоставляет отдельный журнал отмены действий для каждой страницы, измененной в активной записной книжке. Можно отменять неопределенное количество действий на одной странице в течение текущего сеанса редактирования, пока не будет вставлено изображение или внесены изменения другим автором. Перемещение и удаление страниц не может быть отменено. Дополнительные сведения о [OneNote для веб-](https://go.microsoft.com/fwlink/p/?LinkId=272944)сочетаний клавиш.
  
## <a name="video-playback"></a>Воспроизведение видео
<a name="bkmk_Videoplayback"> </a>

Видео-заметки сохраняются в заметках, но не могут быть записаны в OneNote для Интернета. Чтобы воспроизвести видеофайлы, можно загрузить их на свой компьютер. Узнайте больше о [различиях в использовании записной книжки в браузере и в классическом приложении OneNote](https://go.microsoft.com/fwlink/p/?LinkId=272946).
