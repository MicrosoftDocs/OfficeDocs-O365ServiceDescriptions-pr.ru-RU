# <a name="excel-for-the-web"></a>Excel в Интернете

Excel для Интернета (прежнее название — Excel Web App) расширяет возможности Microsoft Excel в веб-браузере, где вы можете работать с книгами непосредственно на веб-сайте, где хранится книга. Все пользователи Office 365 могут просматривать и легко редактировать файлы Office с помощью Office для Интернета.
  
> [!NOTE]
> Книги, размер которых превышает 50 МБ, не могут быть просмотрены в Excel для Интернета в Microsoft SharePoint Online. Для просмотра объемных файлов необходимо установить на компьютер классическое приложение Excel. 

> [!NOTE]
> Для простоты сравнения предложений в список расширенных функций в этой статье включены возможности классического приложения Excel, доступные только в составе набора Office (например, Office профессиональный плюс, Office стандартный или Office 365 профессиональный плюс). Список всех Office для веб-функций, например Word для Интернета и PowerPoint для Интернета, представлен [в статье доступность функций в Office для веб-планов](office-online-service-description.md#feature-availability-across-office-for-the-web-plans).

Чтобы узнать, какие типы файлов поддерживаются в Excel для Интернета, ознакомьтесь [со статьей Поддерживаемые типы файлов для Office в Интернете](office-online-service-description.md#supported-file-types-for-office-for-the-web). Дополнительные сведения о различиях между Excel в Интернете и классическом приложении Excel приведены в статье [различия между использованием книги в браузере и в Excel](https://support.office.com/article/f0dc28ed-b85d-4e1d-be6d-5878005db3b6).
  
## <a name="add-background-color-to-sheet-tabs"></a>Добавление цвета фона на вкладки листа 

Добавьте цвет заливки фона на вкладки листа в нижнюю часть книги, чтобы выделить их. Дополнительную информацию можно узнать [в статье Добавление цвета фона на вкладке листа](https://support.office.com/article/440B28F2-3146-4DCA-95DF-3B9D43ACBE59).

## <a name="add-hyperlinks"></a>Добавление гиперссылок

Добавление гиперссылки в ячейку для связи с веб-страницей или открытия файлов в Интернете. Можно ввести веб-адрес непосредственно в ячейку или воспользоваться командой "Добавить гиперссылку".

## <a name="advanced-data-types-stocks-and-geography"></a>Расширенные типы данных: Stocks и geography

В Excel можно получить биржевые и географические данные. Это так же просто, как ввод текста в ячейку, а затем преобразование его в тип данных Stocks или geography. Узнайте больше о [биржевых и географических типах данных](https://support.office.com/article/e61a33056-9935-484f-8ac8-f1a89e210877).
  
## <a name="advanced-time-filtering-timeline-slicer"></a>Advanced time filtering (Timeline slicer)

Элемент временной шкалы позволяет фильтровать смоделированные данные. Функция обеспечивает удобный просмотр и возможность изменения непрерывного диапазона дат и фильтрации сводных объектов, например сводных таблиц или сводных диаграмм.  
  
## <a name="alignment"></a>Выравнивание

Используйте кнопки "Выравнивание" на вкладке "Главная" для изменения выравнивания текста.  
  
## <a name="apply-conditional-formatting"></a>Применение условного форматирования

Используйте условное форматирование, которое поможет визуально просматривать и анализировать данные, обнаруживать критические проблемы и определять закономерности и тенденции. Эта функция доступна в Excel для Интернета. Другие расширенные функции доступны только в классическом приложении Excel.
  
## <a name="apply-smartrecommended-formatting"></a>Apply smart/recommended formatting

Недоступно в Excel для Интернета. В классическом приложении Excel Рекомендуемые диаграммы позволяют выбирать из разнообразных диаграмм, которые лучше всего подходят для представления данных. Узнайте больше о [том, как использовать рекомендации по работе с диаграммами в Excel](https://support.office.com/article/cd131b77-79c7-4537-a438-8db20cea84c0).
  
## <a name="autocomplete"></a>Автозаполнения

## <a name="autosum"></a>Автосумма

Сведите столбец или строку чисел, выбрав ячейки, которые необходимо суммировать, и дважды щелкните команду "Автосумма". Результат появится в следующей пустой ячейке.

Вы можете переместить значение ячейки в Excel для веба, перетащив ячейки из одного места в другое.

## <a name="draw-cell-borders"></a>Рисование границ ячеек

Можно рисовать границы вокруг ячеек листа, чтобы визуально упорядочить данные. Дополнительные сведения о [границах ячеек в Excel](https://support.office.com/article/dc8a310b-92e3-46a7-9f17-2ab745810f4a#ID0EAADAAA=Web).

Dropbox — служба размещения файлов с облачным хранилищем, синхронизацией файлов, личным облаком и клиентским программным обеспечением.

Сохранив книгу Excel в хранилище OneDrive.com, вы сможете внедрить ее непосредственно на странице блога или веб-сайта. Пользователи смогут сортировать, фильтровать и вычислять данные прямо в книге, а если вы обновите книгу в OneDrive, пользователи увидят последние изменения после того, как обновят страницу. Узнайте больше о [внедрении книги Excel в блог](https://go.microsoft.com/fwlink/p/?LinkId=271673).
  
## <a name="external-references-links"></a>Внешние ссылки

Внешняя ссылка  это ссылка на ячейку или диапазон на листе в другой книге Excel либо ссылка на определенное имя в другой книге. Вы можете использовать Excel для Интернета, чтобы просматривать внешние ссылки, но их нельзя создать с помощью Excel для Интернета. Для создания и обновления внешних ссылок потребуется классическое приложение Excel. 
  
## <a name="fill-handle"></a>Маркер заполнения

Перетяните маркер заполнения, который появится в правом нижнем углу выбранной ячейки или диапазона ячеек, чтобы заполнить соседние ячейки данными.
  
## <a name="find"></a>Найти

Поиск содержимого ячейки на активном листе. Простое в использовании диалоговое окно позволяет выполнить поиск в текущем выделенном фрагменте на листе. **Найти все** позволяет выделять все вхождения искомого термина.
  
## <a name="font-and-cell-formatting"></a>Форматирование шрифта и ячеек

Настройка нужного представления данных. Выберите стиль и цвет шрифта или измените размер и цвет текста. Кроме того, можно рисовать границы ячеек и выбрать цвета границы.
  
## <a name="formula-bar"></a>Formula bar

В строке формул отображаются формулы, по которым вычислены результаты в ячейках. Части формулы можно добавлять, изменять и удалять в браузере так же, как и в классическом приложении Excel. В Excel для веба выделяются аргументы формулы с цветами, которые обеспечивают визуальное сопоставление между формулой и данными на листе.
  
## <a name="formula-tools-advanced"></a>Работа с формулами, дополнительно

Классическое приложение Microsoft Excel поддерживает наиболее расширенные инструменты формул, включая стиль сносок 3D и формулы массивов. Excel для веб-сайта поддерживает только основные формулы Excel. Узнайте больше о [формулах](https://go.microsoft.com/fwlink/p/?LinkId=271665).
  
## <a name="freeze-panes"></a>Freeze panes

Функция "закрепить области" доступна в Excel для веб-ленты. Узнайте больше о [том, как закрепить области в Excel](https://go.microsoft.com/fwlink/p/?LinkId=282331) , и [Ознакомьтесь с советами по работе с Excel для веб-сайта](https://go.microsoft.com/fwlink/?linkid=869873).
  
## <a name="full-fidelity-reading-view"></a>Режим чтения с высокой четкостью

Все данные книги, доступные в браузере в режиме редактирования, можно открыть и в режиме чтения.

Добавьте некоторые визуальные элементы, вставив изображения в электронную таблицу. Узнайте больше о [вставке рисунков в Excel для Интернета](https://support.office.com/article/73660701-018b-4961-bc28-6976cf9ebfb3).

Добавление и форматирование фигур, таких как поля, круги, линии, соединители или стрелки, в Excel для веб-таблицы. Узнайте больше о [добавлении фигур](https://support.office.com/article/0e492bb4-3f91-43b5-803f-dd0998e0eb89#OfficeVersion=Web).
  
## <a name="merge-cells"></a>Merge cells

Use Merge &amp; Center on the Home tab to combine and center the contents of the selected cells in one larger cell. Вы можете изменить выравнивание, нажав кнопки выравнивания.

## <a name="named-ranges"></a>Именованные диапазоны

Вы не можете создавать именованные диапазоны в Excel для Интернета, но вы можете использовать именованные диапазоны, созданные в классической версии Excel, в электронной таблице в Excel для Интернета. При выборе именованного диапазона его имя отображается непосредственно перед строкой формул.

## <a name="number-formatting"></a>Форматирование чисел

Изменение формата чисел. Выберите формат из списка, например "Валюта" или "Краткая дата", измените количество десятичных знаков или вставьте разделитель разрядов.

 Некоторые, но не все надстройки Office доступны в Excel для Интернета. Вы можете приобрести надстройку для Excel в Магазине Office. Узнайте, [как приобрести надстройку Excel](https://go.microsoft.com/fwlink/p/?LinkId=271667).

## <a name="offline-viewing-and-authoring"></a>Просмотр и редактирование в автономном режиме

Excel для Интернета запускается из Интернет-браузера и использует подключение к Интернету. Чтобы иметь доступ к электронным таблицам в автономном режиме, на компьютере необходимо установить классическое приложение Microsoft Excel, которое можно использовать для просмотра и редактирования листов Excel.

## <a name="pivottables"></a>Сводные таблицы

Вы можете вставлять сводные таблицы в Excel для веб-таблицы, вычислять, подсчитывать и анализировать данные. Узнайте больше о том, как вставить [сводную таблицу в Excel для Интернета](https://support.office.com/article/A9A84538-BFE9-40A9-A8E9-F99134456576#OfficeVersion=Web).

## <a name="post-to-social-network-and-present-online"></a>Опубликовать в социальной сети и показать через Интернет

Отправляйте нужные части электронных таблиц через Интернет, внедряя их на своих страницах социальных сетей или в беседах и собраниях в Skype для бизнеса. Excel для Интернета не поддерживает эти функции.
  
## <a name="power-pivot-viewing"></a>Просмотр Power Pivot

Excel для Интернета позволяет просматривать таблицы и диаграммы Power PIVOT, но для создания моделей данных Power Pivot необходимо использовать классическое приложение Excel.
  
## <a name="power-view-viewing"></a>Power View viewing

Вы можете просматривать листы Power View в книге и взаимодействовать с ними, но вы не можете создавать Power View с помощью Excel для Интернета. Создание Power View доступно только с помощью классического приложения. Power View позволяет создавать данные, предоставлять к ним доступ и анализировать их с помощью интерактивных представлений, готовых к презентации. Просматривайте информацию интерактивных карт, диаграмм и таблиц, связанных друг с другом, на одном листе. Узнайте больше о [Power View](https://go.microsoft.com/fwlink/p/?LinkId=271674).
  
## <a name="print"></a>Печать

С помощью команды Print (печать) в Excel веб-сайта можно отправить на принтер весь лист или выделенный фрагмент. Узнайте больше о [том, как печатать в Excel для Интернета](https://go.microsoft.com/fwlink/p/?LinkId=271676).
  
## <a name="real-time-co-authoring"></a>Real-time co-authoring

Несколько пользователей могут одновременно работать с одной электронной таблицей, используя для просмотра веб-браузер, а не классическое приложение Excel. Благодаря обновляемым в реальном времени сведениям о присутствии вы можете видеть, над какой частью документа работают другие авторы (что позволяет избегать конфликтов при редактировании) и какие изменения они вносят. Дополнительные сведения о совместном редактировании в реальном времени см. в видео [Одновременная работа нескольких пользователей над книгами Excel с помощью функции совместного редактирования](https://go.microsoft.com/fwlink/?linkid=850793).
  
## <a name="recommended-chart-creation-and-editing-with-formatting-controls"></a>Создание рекомендованных диаграмм и их редактирование с помощью элементов управления форматированием

Расширенные диаграммы поддерживаются только в классическом приложении Microsoft Excel. Excel для Интернета не поддерживает эти функции. Узнайте больше о [доступных типах диаграмм](https://go.microsoft.com/fwlink/p/?LinkId=271666).

## <a name="rename-and-add-sheets"></a>Rename and add sheets

Выберите значок Новый лист, чтобы добавить лист. Щелкните правой кнопкой мыши ярлычок любого листа, чтобы переименовать его.
  
## <a name="rename-file-while-workbook-is-open"></a>Переименование файла в открытой книге

В Excel для Интернета можно переименовать файл книги, не закрывая его, выбрав имя файла в заголовке и введя новое имя. Чтобы переименовать файл с помощью классического приложения Excel, закройте файл, перейдите к его расположению на устройстве и переименуйте файл.
  
## <a name="replace"></a>Заменить

Недоступно в Excel для Интернета. Если на вашем компьютере установлено классическое приложение Excel, его можно использовать для поиска и замены содержимого в документе. Дополнительные сведения об [использовании функции "Поиск и замена" в Excel](https://go.microsoft.com/fwlink/p/?LinkId=282333).
  
## <a name="rights-management-irm-and-password-security"></a>Управление правами: Управление правами на доступ к данным и защита паролем

 Защищайте свои книги с помощью паролей, разрешений и других ограничений в классическом приложении Excel. В Excel для Интернета можно загружать и взаимодействовать с книгами, содержащими защиту листа, что запрещает пользователям выбирать и вводить текст в защищенных ячейках. ## <a name="vba-and-macro-scripting"></a>VBA и сценарии макросов

Классическое приложение Microsoft Excel является эффективным средством для управления данными, их анализа и представления. Несмотря на богатый выбор возможностей, иногда организациям удобнее использовать язык программирования Visual Basic для приложений (VBA), чтобы создать макрос, который выполняет однообразные повседневные задачи, или выполнить действие, которое отсутствует в пользовательском интерфейсе. Вы не можете создавать макросы в Excel в Excel для Интернета, но вы можете открывать и редактировать электронные таблицы с поддержкой VBA без удаления (или повреждения) VBA, содержащегося в файле. Узнайте больше о том, [как начать работать с макросами](https://go.microsoft.com/fwlink/p/?LinkId=271678).
