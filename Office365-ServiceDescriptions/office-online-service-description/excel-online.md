# <a name="excel-for-the-web"></a>Excel для Интернета

Excel для Интернета (прежнее название — Excel Web App) расширяет возможности Microsoft Excel в веб-браузере, где вы можете работать с книгами непосредственно на веб-сайте, где хранится книга. Все пользователи Office 365 могут просматривать и легко редактировать файлы Office с помощью Office для Интернета.
  
> [!NOTE]
> Книги, размер которых превышает 30 МБ, не могут быть просмотрены в Excel для Интернета в Microsoft SharePoint Online. Для просмотра объемных файлов необходимо установить на компьютер классическое приложение Excel. 
  
> [!NOTE]
> Для простоты сравнения предложений в список расширенных функций в этой статье включены возможности классического приложения Excel, доступные только в составе набора Office (например, Office профессиональный плюс, Office стандартный или Office 365 профессиональный плюс). Список всех Office для веб-функций, например Word для Интернета и PowerPoint для Интернета, представлен [в статье доступность функций в Office для веб-планов](https://docs.microsoft.com/office365/servicedescriptions/office-online-service-description/office-online-service-description). 

Чтобы узнать, какие типы файлов поддерживаются в Excel для Интернета, ознакомьтесь [со статьей Поддерживаемые типы файлов для Office в Интернете](https://docs.microsoft.com/office365/servicedescriptions/office-online-service-description/office-online-service-description). Дополнительные сведения о различиях между Excel в Интернете и классическом приложении Excel приведены в статье [различия между использованием книги в браузере и в Excel](https://support.office.com/article/differences-between-using-a-workbook-in-the-browser-and-in-excel-f0dc28ed-b85d-4e1d-be6d-5878005db3b6?ocmsassetID=HA010369179&CorrelationId=1b10d00b-d045-439b-a3e4-b03be889090e&ui=en-US&rs=en-US&ad=US).
