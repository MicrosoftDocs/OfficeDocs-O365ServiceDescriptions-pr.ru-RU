# <a name="microsoft-project-service-description"></a>Описание службы Microsoft Project

## <a name="overview"></a>Обзор

Microsoft Project предлагает следующие приложения для удовлетворения потребностей организации в управлении проектами и работой:

- Project в Интернете
- Project Online 
- Клиент Project Online для настольных ПК

Эта статья поможет вам понять, какие приложения предоставляются в каждой подписке Project и какие возможности предлагает каждое приложение.

### <a name="project-for-the-web"></a>Project в Интернете

Project в Интернете — это последнее предложение корпорации Майкрософт для облачного управления работой и проектами. Project для Интернета предоставляет простые и мощные возможности управления работой для удовлетворения большинства потребностей и ролей. Руководители проектов и участники группы могут использовать Project в Интернете для планирования работы любого размера и управления ими.

Project для Интернета построен на платформе Microsoft Power Platform. Платформа Power Platform состоит из PowerApps, Power Automate, Power BI и Microsoft Dataverse. Данные Project для Интернета хранятся в microsoft Dataverse. 

Project для Интернета доступен через три подписки: Project Plan 1, Project Plan 3 и Project Plan 5. Функции, перечисленные в разделе ["Функции Project для](#project-for-the-web-features) Интернета", взаимодействуют только с данными в хранилище данных Project для Интернета.

### <a name="project-online"></a>Project Online
 
Project Online — это гибкое интерактивное решение для управления портфелем проектов (PPM) и повседневной работы. Project Online предоставляет мощные возможности управления проектами для планирования, приоритетов и управления проектами и портфелями проектов практически из любого места на любом &mdash; устройстве. Project Online может использоваться администраторами, руководителями и обозревателями портфелей, руководителями проектов и ресурсов, руководителями и участниками групп.

Project Online создан на платформе SharePoint и хранит данные в хранилище данных SharePoint.

Project Online доступен через две подписки: Project Plan 3 и Project Plan 5. Функции, перечисленные в [разделе "Функции Project Online",](#project-online-features) взаимодействуют только с данными в хранилище данных SharePoint Project Online.

Функциональные возможности членов группы для Project Online доступны через подписку на Project Online essentials или Project (план 1). Project Online Essentials —  это подписка на надстройки для членов группы для клиентов, у которых есть подписки на Project (план 3) или Project (план 5). Подписки на Project Online essentials и Project (план 1) дают участникам группы веб-интерфейс для выполнения обновлений задач и времени для проектов, управляемых в Project Online.

Участники группы с подписками на Project Online Essentials или Project Plan 1 могут выполнять следующие функции Project Online:

- Использование веб-интерфейса
- Обновление задач, проблем и рисков
- Отправка расписаний
- Совместное доступ к документам и совместное взаимодействие с пользователями Microsoft Teams или Skype для бизнеса

Project Online Essentials не включает Project в Интернете. Project (план 1) включает Project в Интернете.

### <a name="project-online-desktop-client"></a>Клиент Project Online для настольных ПК

Клиент Microsoft Project Online для настольных ПК — это программа управления проектами, которая включает автоматическое планирование, управление ресурсами проектов и встроенные отчеты. Клиент Project Online для настольных ПК можно использовать в качестве автономных приложений или подключаться к Project Online. Клиент Project Online для настольных ПК — это версия настольного клиента Project Профессиональный по подписке, доступная в подписках Project (план 3) и Project (план 5).

### <a name="using-project-for-the-web-and-project-online-together"></a>Совместное использование Project в Интернете и Project Online

Хотя Project для Интернета и Project Online являются отдельными приложениями, их можно использовать рядом. Вы можете увидеть все проекты из обоих приложений на домашней компьютере Project. Вы можете использовать возможность "План проектов" для создания стратегических карт, включив в себя проекты из обоих приложений. Вы также можете использовать Power BI и пакет содержимого Project для Веб-Power BI, чтобы получить представление о проектах и ресурсах в двух приложениях. <sup>3</sup>

На этой схеме показано, как приложения могут работать вместе друг с другом.

![Схема, на которой показаны project в Интернете и Project Online вместе](../media/project-online-applications.png)

В будущем корпорация Майкрософт сосредоточится на project для интернет-инноваций. Однако вы можете продолжать использовать Project Online с уверенностью, так как он продолжит получать ключевые улучшения производительности и безопасности.

## <a name="service-considerations"></a>Вопросы обслуживания

> [!NOTE]
> Project для Интернета в настоящее время недоступен для GCC, GCC High и DoD. Мы работаем над доставкой Project в Интернете, но в настоящее время не можем делиться сроками.

### <a name="licensing-considerations"></a>Принципы лицензирования

- Для любого взаимодействия на сайте Project Online требуется по крайней мере подписка на Project (план 3) или Project (план 5) в клиенте.
- По истечении срока действия последней подписки на Project (план 1), Project (план 3) или Project (план 5) экземпляры Project для Интернета не будут автоматически удаляться, пока у вас не будет активных подписок, которые зависят от microsoft Dataverse.
- По истечении срока действия последней подписки на Project Plan 3 или Project Plan 5 экземпляры Project Online будут удалены через 120 дней.
- Пробные подписки Project Online будут удалены через 30 дней после окончания пробного периода.
- Пробные экземпляры Project для веб-пробной подписки не будут удалены до тех пор, пока у вас не будет активных подписок, которые зависят от microsoft Dataverse.

### <a name="use-of-sharepoint-online"></a>Использование SharePoint Online

Project Online требует использования SharePoint Online, которая была предусмотрена в Составе Project Online. Права на функции SharePoint Online, предоставляемые подписками на Project (план 3) или Project (план 5), ограничены хранением данных и доступом к ним для поддержки Project Online.

### <a name="project-roadmap-and-power-automate"></a>План проекта и Power Automate

План проекта требует использования Power Automate, который является частью подписки на Project. Права на функции Power Automate ограничены возможностями Power Automate, требуемой в project Roadmap. Функция Power Automate, требуемая в плане развития Project, отображается в  разделе "Приложения" Центра администрирования Microsoft 365 как интеграция данных для Project с планом обслуживания **Flow.**

### <a name="project-for-the-web-and-microsoft-dataverse"></a>Project для Интернета и Microsoft Dataverse

Project для Интернета требует использования microsoft Dataverse для хранения своих данных. База данных Microsoft Dataverse будет создана в рамках подписки На Project. Права на функции Microsoft Dataverse ограничены хранением данных и доступом к ним для поддержки Project в Интернете. Функциональность Microsoft Dataverse, требуемая для Project, отображается в  разделе "Приложения" Центра администрирования Microsoft 365 как общая служба данных **для Project.**

Для клиентов Project с пятью (5) или более веб-лицензиями Project для Интернета клиенты Project для Интернета могут развернуть в средах Power Platform Production и "Песочнице".

### <a name="data-backup-and-retention"></a>Резервное копирование и хранение данных

Project для Интернета и Project Online имеют ту же политику резервного копирования и хранения данных, что и Office 365. Подробные сведения см. в сведениях о [хранении, удалении и уничтожении данных в Office 365.](https://docs.microsoft.com/office365/Enterprise/office-365-data-retention-deletion-and-destruction-overview)
