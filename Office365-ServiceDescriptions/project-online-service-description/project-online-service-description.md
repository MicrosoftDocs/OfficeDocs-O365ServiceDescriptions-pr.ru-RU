# <a name="project-online-service-description"></a>Описание службы Project Online

Project Online — это гибкое оперативное решение для управления портфелем проектов (PPM) и повседневной работы. Project OnlineOffice 365: позволяет планировать проекты и инвестиции портфеля проектов, определять их приоритеты и управлять ими практически где угодно и с любого устройства. Project Online может использоваться администраторами, руководителями и обозревателями портфелей, руководителями проектов и ресурсов, руководителями и участниками групп. 
  
Доступно три плана Project Online: базовый, профессиональный и расширенный.
  
Сравнение функций в разных планах см. в статье [Сравнение планов Project Online](https://products.office.com/en-us/project/compare-microsoft-project-management-software).
  
Вы можете экспортировать, сохранять и печатать страницы описаний служб Office 365. Узнайте, как [экспортировать несколько страниц](https://go.microsoft.com/fwlink/?LinkId=403349).
  
## <a name="project-online-boundaries-and-limits"></a>Ограничения Project Online

Служба Project Online имеет некоторые ограничения. Для получения дополнительных сведений ознакомьтесь [с ограничениями и границами программного обеспечения Project Online](https://support.office.com/en-us/article/Project-Online-software-boundaries-and-limits-5A09DBCE-1E68-4A7B-B099-D5F1B21BA489).
  
## <a name="data-backup"></a>Резервное копирование данных

 Project Online имеет ту же политику резервного копирования и хранения данных, что и Office 365. Подробные сведения см. [в статье хранение, удаление и уничтоженИе данных в Office 365](https://docs.microsoft.com/office365/securitycompliance/office-365-data-retention-deletion-and-destruction-overview).
  
## <a name="use-of-sharepoint-online"></a>Использование SharePoint Online

Служба Project Online предусматривает использование службы SharePoint Online, которая будет предоставлена вместе со службой Project Online. Права на функции SharePoint Online, предоставляемые Project Online Professional или Premium, ограничены хранением и доступом к данным для поддержки Project Online.
  
## <a name="licensing-considerations"></a>Принципы лицензирования

### <a name="internal-users"></a>Внутренние пользователи

Для любого взаимодействия на сайте Project Online требуется лицензия на проект Online.
  
- Требуется по крайней мере Project Online базовый.
    
- Для более сложных вариантов использования пользователям необходимо иметь соответствующие лицензии. Руководителям проектов требуется по крайней мере лицензия Project Online профессиональный. Для руководителей портфелей требуется лицензия Project Online Premium.
    
> [!IMPORTANT]
>  Project Online Essentials — это лицензия *на надстройку* члена группы для клиентов, у которых есть Project Online профессиональный или Project Online Premium. 
<br/>После истечения срока действия последней лицензии Project Online Professional или Project Online Premium экземпляры Project Web App будут удалены по истечении 120 дней. 
<br/>Для пробных подписок на Project Online экземпляры Project Web App будут удалены через 30 дней после завершения пробного периода. 
  
### <a name="external-users"></a>Внешние пользователи

Лицензирование Project Online для внешних пользователей следует той же политике, что и для внутренних пользователей. Для любого взаимодействия на сайте Project Online требуется лицензия на проект Online.
